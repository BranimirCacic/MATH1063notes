Lemma 1.1.9.

Let \(N\) be a natural number, let \(0 \leq k \leq N-1\text{,}\) and let \(U_{k;N}\) be defined as in Lemma 1.1.7. Then \(\frac{k^2}{N^3} \leq \Area(U_{k;N}) \leq \frac{(k+1)^2}{N^3}\text{.}\)

Proof.

Again, I’ll be a little sketchy with this proof since it concerns the underlying geometry of this problem, which is intuitively clear but a little finnicky to write out in complete detail.
Recall that \(a^2 \leq b^2\) whenever \(0 \leq a \leq b\text{.}\) Thus, it follows that \(\frac{k^2}{N^2} \leq x^2 \leq \frac{(k+1)^2}{N^2}\) whenever \(\frac{k}{N} \leq x \leq \frac{k+1}{N}\text{.}\) Hence, it follows that \(U_{k;N}\) contains the smaller rectangle
\begin{equation*} R_1 \defn \set{(x,y) \in \R^2}{\frac{k}{N} \leq x \leq \frac{k+1}{N}, \, 0 \leq y \leq \frac{k^2}{N^2}} \end{equation*}
and is contained by the larger rectangle
\begin{equation*} R_2 \defn \set{(x,y) \in \R^2}{\frac{k}{N} \leq x \leq \frac{k+1}{N}, \, 0 \leq y \leq \frac{(k+1)^2}{N^2}}\text{,} \end{equation*}
so that \(\Area(R_1) \leq \Area(U_{k;N}) \leq \Area(R_2)\) by Item 4 of Assumption 1.1.4.
On the one hand, since \(R_1\) is a rectangle with base \(\frac{1}{N}\) and height \(\frac{k^2}{N^2}\text{,}\) it follows that \(\Area(R_1) = \frac{k^2}{N^3}\) by Item 1 of Assumption 1.1.4. On the other hand, since \(R_2\) is a rectangle with base \(\frac{1}{N}\)L and height \(\frac{(k+1)^2}{N^2}\text{,}\) it follows that \(\Area(R_2) =\frac{(k+1)^2}{N^3}\) by Item 1 of Assumption 1.1.4. Thus, at last, it follows that
\begin{equation*} \frac{k^2}{N^3} = \Area(R_1) \leq \Area(U_{k;N}) \leq \Area(R_2) = \frac{(k+1)^2}{N^3}. \end{equation*}
