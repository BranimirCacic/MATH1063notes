Lemma 1.1.11.

For every natural number \(N\text{,}\)
\begin{equation*} \frac{1}{N^3}(1^2 + 2^2 + \cdots (N-1)^2) \leq \Area(U) \leq \frac{1}{N^3}(1^2 + 2^2 + \cdots N^2)\text{.} \end{equation*}

Proof.

Let \(M\) be a natural number. On the one hand, \(\Area(U) = \Area(U_{0;N}) + \cdots + \Area(U_{k;N})\) by Lemma 1.1.7. On the other hand, \(\frac{k^2}{N^3} \leq \Area(U_{k;N}) \leq \frac{(k+1)^2}{N^3}\) for each \(0 \leq k \leq N-1\) by Lemma 1.1.9. Thus, we conclude that
\begin{align*} \Area(U) \amp = \Area(U_{0,1/N}) + \Area(U_{1/N,2/N}) + \cdots + \Area(U_{(N-1)/N,1})\\ \amp \geq \frac{0^2}{N} + \frac{1^2}{N} + \cdots \frac{(N-1)^2}{N} \\ \amp = \frac{1}{N}\left(1^2 + 2^2 + \cdots + (N-1)^2\right) \end{align*}
and
\begin{align*} \Area(U) \amp = \Area(U_{0,1/N}) + \Area(U_{1/N,2/N}) + \cdots + \Area(U_{(N-1)/N,1}) \\ \amp \leq \frac{1^2}{N} + \frac{2^2}{N} + \cdots \frac{N^2}{N}\\ \amp = \frac{1}{N}\left(1^2 + 2^2 + \cdots + N^2\right) \text{.} \end{align*}
