Example 1.7.5.

Let \(n\) be a non-negative integer. On the one hand, define a function \(f\) on \(\R\) by \(f(x) \defn x^n\text{.}\) On the other hand, define a function \(F\) on \(\R\) by \(F(x) \defn \frac{1}{n+1}x^{n+1}\text{.}\) Then \(F\) is an antiderivative of the continuous function \(f\) since
\begin{equation*} F^\prime(x) = \ddx\left(\frac{1}{n+1}x^{n+1}\right) = \frac{1}{n+1} \cdot (n+1)x^{(n+1)-1} = x^n = f(x) \end{equation*}
with the traditional harmless abuses of notation. Hence, for all \(a \lt b\text{,}\) it follows by Theorem 1.7.3 that
\begin{equation*} \int_a^b f = F(b) - F(a) = \frac{1}{n+1}b^{n+1} - \frac{1}{n+1}a^{n+1} = \frac{b^{n+1}-a^{n+1}}{n+1}\text{;} \end{equation*}
the same calculation in Lebiniz notation looks like
\begin{equation*} \int_a^b x^n \du{x} = \eval{\frac{1}{n+1}x^{n+1}}{a}{b} = \frac{1}{n+1}b^{n+1} - \frac{1}{n+1}a^{n+1} = \frac{b^{n+1}-a^{n+1}}{n+1}\text{.} \end{equation*}
Note that setting \(N=2\text{,}\) \(a = 0\text{,}\) and \(b=1\) effortlessly recovers the calculation \(\int_0^1 x^2 \du{x} = \frac{1}{3}\) of Example 1.4.4.
