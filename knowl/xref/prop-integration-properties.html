Proposition 1.4.11. Basic properties of integration.

  1. Suppose that \(f\) is integrable on \([a,b]\) and that \(a \lt c \lt b\text{.}\) Then \(f\) is integrable on both \([a,c]\) and \([c,b]\) and
    \begin{equation*} \int_a^b f = \int_a^c f + \int_c^b f\text{,} \end{equation*}
    i.e., \(\int_a^b f(x)\du{x} = \int_a^c f(x)\du{x} + \int_c^b f(x)\du{x}\text{.}\)
  2. Suppose that \(f\) and \(g\) are both integrable on \([a,b]\text{.}\) Define \(f+g\) on \([a,b]\) by \((f+g)(x) \defn f(x) + g(x)\text{.}\) Then \(f+g\) is integrable on \([a,b]\) and
    \begin{equation*} \int_a^b (f+g) = \int_a^b f + \int_a^b g \end{equation*}
    i.e., \(\int_a^b \left(f(x)+g(x)\right)\du{x} = \int_a^b f(x)\du{x} + \int_a^b g(x)\du{x}\text{.}\)
  3. Suppose that \(f\) is integrable on \([a,b]\text{;}\) let \(c\) be a real number. Define \(cf\) on \([a,b]\) by \((cf)(x) \defn cf(x)\text{.}\) Then \(cf\) is integrable on \([a,b]\) and
    \begin{equation*} \int_a^b cf = c \int_a^b f \end{equation*}
    i.e., \(\int_a^b c f(x) \du{x} = c \int_a^b f(x)\du{x}\text{.}\)

Proof.

I’ll give the proof of Item 2 as a typical example of how these kinds of facts are proved (and an illustration of how abjectly tedious the necessary bookkeeping is). In what follows, if \(h\) is an integrable function on \([a,b]\) and \(P = (x_0,x_1,\dotsc,x_N)\) is a partition of \([a,b]\text{,}\) let
\begin{align*} m_k[h,P] \amp \defn \inf\set{h(x)}{x_{k_1}\leq x \leq x_k}\\ M_k \amp \defn \sup\set{h(x)}{x_{k_1}\leq x \leq x_k} \end{align*}
for each \(1 \leq k \leq N\text{,}\) so that
\begin{align*} L(h,P) \amp = \sum_{k=1}^N m_k[h,P](x_k-x_{k-1}) \\ U(h,P) \amp = \sum_{k=1}^N M_k[h,P] (x_k-x_{k-1})\text{.} \end{align*}
Let’s first prove that \(f+g\) is integrable. Let \(\epsilon \gt 0\) be given. On the one hand, by Theorem 1.4.6 applied to \(f\text{,}\) there exists a partition \(P_1\) of \([a,b]\) such that \(U(f,P_1) - L(f,P_1) \lt \frac{\epsilon}{2}\text{.}\) On the other hand, by Theorem 1.4.6 applied to \(g\text{,}\) there exists a partition \(P_2\) of \([a,b]\) such that \(U(g,P_2) - L(f,P_2) \lt \frac{\epsilon}{2}\text{.}\) Hence, let \(P=(x_0,x_1,\dotsc,x_N)\) be the partition of \([a,b]\) obtained by merging the lists \(P_1\) and \(P_2\text{,}\) so that, by Item 2,
\begin{align*} L(f,P) \amp \geq L(f,P_1) \amp U(f,P) \amp \leq U(f,P_1)\\ L(g,P) \amp \geq L(g,P_2) \amp U(g,P) \amp \leq U(g,P_2)\text{.} \end{align*}
Now, for each \(1 \leq k \leq N\text{,}\) observe that
\begin{equation*} m_k[f,P]+m_k[g,P] \leq f(x) + g(x) \leq M_k[f,P]+M_k[g,P] \end{equation*}
for all \(x_{k-1} \leq x \leq x_k\text{,}\) so that
\begin{align*} m_k[f,P] + m_k[g,P] \amp \leq m_k[f+g,P]\\ M_k[f,P] + M_k[g,P] \amp \geq M_k[f+g,P]\text{.} \end{align*}
Hence,
\begin{align*} L(f+g,P) \amp = \sum_{k=1}^N m_k[f+g,P](x_k-x_{k-1}) \\ \amp \geq \sum_{k=1}^N \left(m_k[f,P]+m_k[g,P]\right)(x_k-x_{k-1}) \\ \amp = \sum_{k=1}^N m_k[f,P](x_k-x_{k-1}) + \sum_{k=1}^N m_k[g,P](x_k-x_{k-1})\\ \amp = L(f,P) + L(g,P) \\ \amp \geq L(f,P_1) + L(g,P_2) \text{.} \end{align*}
Similarly, one can show that \(U(f+g,P) \leq U(f,P_1) + U(g,P_2)\text{.}\) Thus, at last,
\begin{align*} U(f+g,P) - L(f+g,P) \amp \leq \left(U(f,P_1) + U(g,P_2)\right) - \left(L(f,P_1) + L(g,P_2)\right) \\ \amp = \left(U(f,P_1)-L(f,P_1)\right) + \left(U(g,P_2) - L(g,P_2)\right) \\ \amp \lt \frac{\epsilon}{2} + \frac{\epsilon}{2}\\ \amp = \epsilon \text{.} \end{align*}
Let’s now show that \(\int_a^b(f+g) = \int_a^b f + \int_a^b g\text{.}\) Let \(\epsilon \gt 0\) be given. By our constructions above, there exists a partition \(P\) of \([a,b]\) such that
\begin{align*} U(f,P) - L(f,P) \amp \lt \frac{\epsilon}{4} \amp U(g,P) - L(g,P) \amp \lt \frac{\epsilon}{4} \text{,} \end{align*}
and moreover,
\begin{align*} L(f,P) + L(g,P) \amp \leq L(f+g,P) \amp U(f+g,P) \amp \leq U(f,P) + U(g,P)\text{.} \end{align*}
On the one hand,
\begin{equation*} L(f,P) + L(g,P) \leq \int_a^b (f+g) \leq U(f,P) + U(g,P) \end{equation*}
since
\begin{equation*} L(f,P) + L(g,P) \leq L(f+g,P) \leq \Lint{a}{b}(f+g) = \int_a^b (f+g) \end{equation*}
and
\begin{equation*} U(f,P) + U(g,P) \geq U(f+g,P) \geq \Uint{a}{b}(f+g) = \int_a^b(f+g)\text{.} \end{equation*}
On the other hand,
\begin{equation*} L(f,P) + L(g,P) \leq \int_a^b f + \int_a^b g \leq U(f,P) + U(g,P) \end{equation*}
since
\begin{equation*} L(f,P) + L(g,P) \leq \Lint{a}{b}f + \Lint{a}{b} g = \int_a^b f + \int_a^b g \end{equation*}
and
\begin{equation*} U(f,P) + U(g,P) \geq \Uint{a}{b}f + \Uint{a}{b}g = \int_a^b f = \int_a^b g. \end{equation*}
Thus, we find that
\begin{align*} 0 \amp \leq \left(\int_a^b (f+g)) \right) - \left(L(f,P)+L(g,P)\right) \\ \amp \leq \left(U(f,P)+U(g,P)\right) - \left(L(f,P)+L(g,P)\right) \\ \amp = \left(U(f,P)-L(f,P)\right) + \left(U(g,P) - L(g,P)\right) \\ \amp = \frac{\epsilon}{4} + \frac{\epsilon}{4} \\ \amp = \frac{\epsilon}{2}\text{,} \end{align*}
and similarly
\begin{equation*} 0 \leq \left(\int_a^b f + \int_a^b g \right) - \left(L(f,P)+L(g,P)\right) \lt \frac{\epsilon}{2}\text{.} \end{equation*}
Hence, at last, by the triangle inequality,
\begin{align*} \abs{\left(\int_a^b f + \int_a^b g \right) - \int_a^b(f+g)} \amp \leq \abs{\left(\int_a^b f + \int_a^b g \right) - \left(L(f,P)+L(g,P)\right)}\\ \amp \quad \abs{\int_a^b (f+g) - \left(L(f,P)+L(g,P)\right)}\\ \amp \leq \frac{\epsilon}{2} + \frac{\epsilon}{2} \\ \amp =\epsilon\text{.} \end{align*}
Since \(\epsilon \gt 0\) was arbitrary, it follows that \(\int_a^b f + \int_a^b g = \int_a^b(f+g)\text{.}\)
