Lemma 1.6.3.

Let \(f\) be an integrable function on a closed interval \([a,b]\text{.}\) Let \(m\) and \(M\) be real numbers such that \(m \leq f(x) \leq M\) for all \(a \leq x \leq b\text{.}\) Then
\begin{equation*} m(b-a) \leq \int_a^b f \leq M(b-a)\text{.} \end{equation*}

Proof.

Consider the trivial partition \(P_1 = (x_0,x_1)\text{,}\) where \(x_0 \defn a\) and \(x_1 \defn b\text{.}\) On the one hand, since \(m\) is a lower bound for \(\set{f(x)}{a \leq x \leq b}\text{,}\) it follows that \(m \leq m_1\text{,}\) hence
\begin{equation*} m(b-a) \leq m_1(b-a) = L(f,P_1) \leq \Lint{a}{b}f = \int_a^b f\text{.} \end{equation*}
On the other hand, since \(M\) is an upper bound for \(\set{f(x)}{a \leq x \leq b}\text{,}\) it follows that \(M_1 \leq M\text{,}\) hence
\begin{equation*} \int_a^b f = \Uint{a}{b}f \leq L(f,P_1) = M_1(b-a) \leq M(b-a)\text{.} \end{equation*}
