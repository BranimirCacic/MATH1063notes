Assumption 1.1.4.

One can give a formal definition
 6 
en.wikipedia.org/wiki/Area#Formal_definition
of the area of planar regions, which is realized by a construction called Jordan content. We’ll assume that all planar regions under consideration have a well-defined area. We’ll also assume without proof the following basic facts about area.
  1. If \(R\) is a rectangle with base \(b\) and height \(h\text{,}\) then \(\Area(R) = bh\text{.}\) In particular, a line segment (rectangle with base or height equal to \(0\)) has area equal to \(0\text{.}\)
  2. If \(A\) is empty, then \(\Area(A) = 0\text{.}\)
  3. Let \(A_1,\dotsc,A_N\) be planar regions with well-defined area. Suppose that \(\Area(A_i \cap A_j) = 0\) for all \(1 \leq i,j \leq N\) satisfying \(i \neq j\text{.}\) Then
    \begin{equation*} \Area(A_1 \cup \cdots \cup A_N) = \Area(A_1) + \cdots + \Area(A_N)\text{.} \end{equation*}
  4. Let \(A\) and \(B\) be planar regions with well-defined area. If \(A \subseteq B\text{,}\) then \(\Area(A) \leq \Area(B)\text{.}\)
