Proposition 1.3.10.

Let \(f\) be a bounded real-valued function on a closed interval \([a,b]\text{.}\)
  1. Let \(P\) be a partition of \([a,b]\text{.}\) Then \(L(f,P) \leq U(f,P)\text{.}\)
  2. Let \(P\) and \(P'\) be partitions of \([a,b]\text{.}\) Suppose that \(P'\) can be obtained from \(P\) by including additional points (i.e., \(P'\) is a refinement of \(P\)). Then
    \begin{equation*} L(f,P) \leq L(f,P') \leq U(f,P') \leq U(f,P)\text{.} \end{equation*}
  3. Let \(P\) and \(P'\) be partitions of \([a,b]\text{.}\) Then \(L(f,P) \leq U(f,P')\text{.}\)

Proof.

Let us first prove Item 1 Let \(P = (x_0,x_1,\dotsc,x_N)\) be a partition of \([a,b]\text{.}\) Given \(1 \leq k \leq N\text{,}\) since
\begin{equation*} m_k = \inf\set{f(x)}{x_{k-1} \leq x \leq x_k} \leq f(x_{k-1}) \leq \sup\set{f(x)}{x_{k-1}\leq x\leq x_k} = M_k\text{,} \end{equation*}
it follows that \(m_k \leq M_k\text{.}\) Thus,
\begin{equation*} L(f,P) = \sum_{k=1}^N m_k (x_k-x_{k-1}) \leq \sum_{k=1}^N M_k(x_k-x_{k-1}) = U(f,P)\text{.} \end{equation*}
Next, let us sketch the proof of Item 2. The general case is a tedious application of the special case where \(P = (a,b)\) and where \(P' = (x_0^\prime,\dotsc,x_N^\prime)\) is any other partition. We’ll show that \(U(f,P') \leq U(f,P)\text{;}\) the argument that \(L(f,P) \leq L(f,P')\) will be qualitatively identical.
For each \(1 \leq k \leq N\text{,}\) let \(M_k^\prime \defn \sup\set{f(x)}{x_{k-1}^\prime \leq x \leq x_k^\prime}\text{,}\) so that \(U(f,P') = \sum_{k=1}^N M_k^\prime(x_k^\prime-x_{k-1}^\prime)\text{;}\) let \(M \defn \sup{f(x)}{a\leq x \leq b}\text{,}\) so that \(U(f,P) = M(b-a)\text{.}\) Given \(1 \leq k \leq \text{,}\) since \(f(x) \leq \sup{f(x)}{a\leq x \leq b} = M\) for all \(x \in [x_{k-1}^\prime,x_k^\prime]\text{,}\) it follows that
\begin{align*} U(f,P') \amp \defn \sum_{k=1}^N M_k^\prime(x_k^\prime-x_k-1^\prime) \\ \amp \leq \sum_{k=1}^N M(x_k^\prime-x_{k-1}^\prime) \\ \amp = M\sum_{k=1}^N(x_k^\prime-x_{k-1}^\prime)\\ \amp = M(b-a) \\ \amp = U(f,P) \text{.} \end{align*}
Finally, let us prove Item 3. Let \(P\) and \(P'\) be partitions of \([a,b]\text{.}\) Let \(P^{\prime\prime}\) be the partition of \([a,b]\) obtained by merging the lists \(P\) and \(P'\text{.}\) Then \(P^{\prime\prime}\) is a refinement of both \(P\) and \(P'\text{,}\) so that
\begin{equation*} L(f,P) \leq L(f,P^{\prime\prime}) \leq U(f,P^{\prime\prime}) \leq U(f,P) \end{equation*}
by Item 1 and Item 2.
