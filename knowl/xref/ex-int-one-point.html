Example 1.4.8.

Define a function \(f\) on \([0,1]\) by
\begin{equation*} f(x) \defn \begin{cases}1 \amp \text{if } x = 1\\ 0 \amp \text{else}\end{cases}\text{.} \end{equation*}
Then \(f\) is integrable on \([0,2]\) and \(\int_0^2 f = 0\text{.}\)
Let’s first show that \(f\) is integrable using Theorem 1.4.6. Let \(0 \lt \epsilon \lt 4\) be given. Define a partition \(P_\epsilon = (x_0,x_1,x_2,x_3)\) of \([0,2]\) by
\begin{align*} x_0 \amp \defn 0 \amp x_1 \amp \defn 1-\frac{\epsilon}{4} \amp x_2 \amp \defn 1+\frac{\epsilon}{4} \amp x_3 \amp \defn 2 \text{.} \end{align*}
On the one hand, on both \([x_0,x_1]\) and \([x_2,x_3]\text{,}\) the function \(f\) has the constant value \(0\text{,}\) so that \(m_1 = 0 = M_1\) and \(m_3 = 0 = M_3\text{.}\) On the other hand, on \([x_1,x_2]\text{,}\) the function \(f\) attains a minimum of \(0\) and a maximum of \(1\text{,}\) so that \(m_2 = 0\) and \(M_2 = 1\text{.}\) Hence,
\begin{equation*} L(f,P_\epsilon) = \sum_{k=1}^3 m_k (x_k-x_{k-1}) = \sum_{k=1}^3 0(x_k-x_{k-1}) = 0 \end{equation*}
and
\begin{align*} U(f,P_\epsilon) \amp = \sum_{k=1}^3 M_k (x_k-x_{k-1})\\ \amp = 0 \cdot ((1-\tfrac{\epsilon}{4})-1) + 1 \cdot ((1+\tfrac{\epsilon}{4})-(1-\tfrac{\epsilon}{4})) + 0 \cdot (2-(1+\tfrac{\epsilon}{4})) \\ \amp = \frac{\epsilon}{2} \text{,} \end{align*}
so that
\begin{equation*} U(f,P_\epsilon) - L(f,P_\epsilon) = \frac{\epsilon}{2}-0 = \frac{\epsilon}{2} \lt \epsilon. \end{equation*}
Since we can do this for all sufficienly small \(\epsilon \gt 0\text{,}\) it follows that \(f\) is integrable on \([0,2]\text{.}\)
Let us now compute \(\int_0^2 f\text{.}\) By our computations above, for every \(0 \lt \epsilon \lt 4\text{,}\)
\begin{equation*} 0 = L(f,P_\epsilon) \leq \Lint{0}{2}f = \int_0^2 f = \Uint{0}{2} f \leq U(f,P_\epsilon) = \frac{\epsilon}{2} \lt \epsilon\text{,} \end{equation*}
so that \(0 \leq \int_0^2 f \lt \epsilon\) for all sufficiently small \(\epsilon \gt 0\text{.}\) Thus, \(\int_0^2 f = 0\text{.}\)
