Definition 1.3.6.

Let \(f\) be a bounded real-valued function on a closed interval \([a,b]\text{,}\) let \(P = (x_0,x_1,\dotsc,x_N)\) be a partition of \([a,b]\text{,}\) and set
\begin{align*} m_k \amp \defn \inf\set{f(x)}{x_{k-1} \leq x \leq x_k} \amp \\ M_k \amp \defn \sup\set{f(x)}{x_{k-1} \leq x \leq x_k} \end{align*}
for each \(1 \leq k \leq N\text{.}\)
  1. The lower sum of \(f\) with respect to \(P\) is
    \begin{equation*} L(f,P) \defn \sum_{k=1}^N m_k(x_k-x_{k_1})\text{.} \end{equation*}
  2. The upper sum of \(f\) with respect to \(P\) is
    \begin{equation*} U(f,P) \defn \sum_{k=1}^N M_k(x_k-x_{k_1})\text{.} \end{equation*}
