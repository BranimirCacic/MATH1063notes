    Section 1.7 The Second Fundamental Theorem of Calculus

    In this section, we discuss the second fundamental theorem of calculus, which gives us an easy way to compute an integral \(\int_a^b f\) when we happen to know (by hook or by crook!) that \(f = F^\prime\) for some function \(F\text{.}\) This remarkable fact therefore lets us leverage everything we know about differentiation to compute integrals of familiar functions (and some not-so-familiar functions too). Our account in this section and Section 1.6 provides a rigorous complement to Section 5.5

    of APEX Calculus and Section 5.7 of Strang’s Calculus.

    Definition 1.7.1.

    Let \(f\) be a function on a closed interval \([a,b]\text{.}\) An antiderivative of \(f\) on \([a,b]\) is a differentiable function \(F\) on \([a,b]\) such that \(F^\prime = f\text{.}\)
    A straightforward application of the first fundamental theorem of calculus (Theorem 1.6.4) yields an important special case where an antiderivative is not only guaranteed to exist but allows us to compute the relevant integral.

    Proposition 1.7.2.

    Let \(f\) be a continuous function on a closed interval \([a,b]\text{;}\) let \(C\) be a real number. Then \(f\) has a unique antiderivative \(F\) on \([a,b]\) such that \(F(a) = C\text{.}\) Moreover, this antiderivative \(F\) satisfies \(\int_a^b f = F(b)-F(a)\text{.}\)

    Proof.

    Note that \(f\) is integrable on \([a,b]\) by Theorem 1.6.6. Hence, by Theorem 1.6.4, the function \(F\) on \([a,b]\) defined by \(F(x) \defn \int_a^b f + C\) is differentiable with derivative \(f + 0 = f\text{;}\) moreover, by construction, it satisfies \(F(a) = \int_a^a f + C = 0 + C = C\text{.}\) Thus, in particular,
    \begin{equation*} F(b) - F(a) = (\int_a^b f + C) - (\int_a^a f + C) = \int_a^b f\text{.} \end{equation*}
    Now, suppose that \(G\) is a differentiable function on \([a,b]\) such that \(G^\prime = f\) and \(G(a)=C\text{.}\) Then \((F-G)^\prime = F^\prime - G^\prime = f - f = 0\text{,}\) so that \(F-G\) is constant. Hence, \(F(x)-G(x) = F(a)-G(a) = C-C = 0\) for all \(a \leq x \leq b\text{.}\)
    The above proposition is good enough for most purposes, but we can actually do better than requiring continuity of \(f\text{.}\) In fact, it’s enough to find (by hook or by crook!) one antiderivative of \(f\text{.}\) This is the content of the second fundamental theorem of calculus.

    Theorem 1.7.3. Second fundamental theorem of calculus.

    Let \(f\) be an integrable function on a closed interval \([a,b]\text{.}\) Suppose that \(F\) is an antiderivative of \(f\) on \([a,b]\text{.}\) Then \(\int_a^b f = F(b)-F(a)\text{.}\)

    Proof.

    It suffices to show that \(L(f,P) \leq F(b)-F(a) \leq U(f,P)\) for every partition \(P\) of \([a,b]\text{.}\) Indeed, suppose we have shown this. On the one hand, since \(F(b)-F(a)\) is an upper bound for every lower sum of \(f\) on \([a,b]\text{,}\) it follows that \(\int_a^b f = \Lint{a}{b}f \leq F(b)-F(a) \text{.}\) On the other hand, since \(F(b)-F(a)\) is a lower bound for every upper sum of \(f\) on \([a,b]\text{,}\) it follows that \(F(b)-F(a) \leq \Uint{a}{b}f = \int_a^b f \text{.}\) Hence, it follows that \(\int_a^b f \leq F(b)-F(a) \leq \int_a^b f\text{,}\) so that \(\int_a^b f = F(b)-F(a)\text{.}\)
    Let \(P = (x_0,x_1,\dotsc,x_N)\) be a partition of \([a,b]\text{.}\) Let \(1 \leq k \leq N\text{.}\) By the mean value theorem

    opentext.uleth.ca/apex-calculus/sec_mvt.html#thm_mvt
    applied to \(F\) on \([x_{k-1},x_k]\text{,}\) there exists \(x_{k-1} \lt x_k^\ast \lt x_k\) such that \(F(x_k)-F(x_{k-1}) = f(x_k^\ast)(x_k-x_{k-1})\text{.}\) Then \(m_k \leq f(x_k^\ast) \leq M_k\) by definition of \(m_k\) and \(M_k\) (see Definition 1.3.6), so that
    \begin{equation*} m_k(x_k-x_{k-1}) \leq f(x_k^\ast)(x_k-x_{k-1}) \leq M_k(x_k-x_{k-1})\text{,} \end{equation*}
    and hence
    \begin{equation*} m_k(x_k-x_{k-1}) \leq F(x_k) - F(x_{k-1}) \leq M_k(x_k-x_{k-1}) \end{equation*}
    Thus, by telescoping summation,
    \begin{equation*} L(f,P) = \sum_{k=1}^N m_k(x_k-x_{k-1}) \leq \sum_{k=1}^N F((x_k)-F(x_{k-1})) = F(b) - F(a) \end{equation*}
    and
    \begin{equation*} L(f,P) = \sum_{k=1}^N M_k(x_k-x_{k-1}) \geq \sum_{k=1}^N F((x_k)-F(x_{k-1})) = F(b) - F(a) \end{equation*}

    Remark 1.7.4.

    The conclusion of Theorem 1.7.3 can be written in Leibniz notation as \(\int_a^b f(x) \du{x} = F(b)-F(a)\) or as \(\int_a^b f(x) \du{x} = F(x)\vert_a^b\text{.}\)

    Example 1.7.5.

    Let \(n\) be a non-negative integer. On the one hand, define a function \(f\) on \(\R\) by \(f(x) \defn x^n\text{.}\) On the other hand, define a function \(F\) on \(\R\) by \(F(x) \defn \frac{1}{n+1}x^{n+1}\text{.}\) Then \(F\) is an antiderivative of the continuous function \(f\) since
    \begin{equation*} F^\prime(x) = \ddx\left(\frac{1}{n+1}x^{n+1}\right) = \frac{1}{n+1} \cdot (n+1)x^{(n+1)-1} = x^n = f(x) \end{equation*}
    with the traditional harmless abuses of notation. Hence, for all \(a \lt b\text{,}\) it follows by Theorem 1.7.3 that
    \begin{equation*} \int_a^b f = F(b) - F(a) = \frac{1}{n+1}b^{n+1} - \frac{1}{n+1}a^{n+1} = \frac{b^{n+1}-a^{n+1}}{n+1}\text{;} \end{equation*}
    the same calculation in Lebiniz notation looks like
    \begin{equation*} \int_a^b x^n \du{x} = \frac{1}{n+1}x^{n+1} \vert_a^b = \frac{1}{n+1}b^{n+1} - \frac{1}{n+1}a^{n+1} = \frac{b^{n+1}-a^{n+1}}{n+1}\text{.} \end{equation*}
    Note that setting \(N=2\text{,}\) \(a = 0\text{,}\) and \(b=1\) effortlessly recovers the calculation \(\int_0^1 x^2 \du{x} = \frac{1}{3}\) of Example 1.4.4.
