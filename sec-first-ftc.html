    Section 1.6 The First Fundamental Theorem of Calculus

    In this section, we’ll prove the first of the two fundamental theorems of calculus, the mathematical breakthrough of the 17th century that links the theory of integration with the theory of differentiation and permits the efficient calculation of integrals in a wide range of cases. In particular, the first fundamental theorem of calculus uses integration to take a given function \(f\) and reverse-engineer a new function \(F\) such that \(F^\prime = f\text{.}\) Our account in this section and Section 1.7 provides a rigorous complement to Section 5.5
     1 
    opentext.uleth.ca/apex-calculus/sec_FTC.html
    of APEX Calculus and Section 5.7 of Strang’s Calculus.
    We begin by showing that integration can be used to construct a continuous function \(F\) from an integrable function \(f\text{,}\) no matter how discontinuous \(f\) may be. When \(f\) is actually continuous, this function \(F\) will turn out to satisfy \(F^\prime = f\text{.}\)

    Theorem 1.6.1.

    Let \(f\) be an integrable function on a closed interval \([a,b]\text{.}\) Let \(F\) be the function \([a,b]\) given by \(F(x) \defn \int_a^x f\text{.}\) Then \(F\) is continuous on \([a,b]\)

    Remark 1.6.2.

    Given a function \(f\) defined at a point \(a\text{,}\) we set \(\int_a^a f \defn 0\text{.}\) Thus, in the context of Theorem 1.6.1, we have \(F(a) \defn \int_a^a f \defn 0\text{.}\)

    Lemma 1.6.3.

    Let \(f\) be an integrable function on a closed interval \([a,b]\text{.}\) Let \(m\) and \(M\) be real numbers such that \(m \leq f(x) \leq M\) for all \(a \leq x \leq b\text{.}\) Then
    \begin{equation*} m(b-a) \leq \int_a^b f \leq M(b-a)\text{.} \end{equation*}

    Proof.

    Consider the trivial partition \(P_1 = (x_0,x_1)\text{,}\) where \(x_0 \defn a\) and \(x_1 \defn b\text{.}\) On the one hand, since \(m\) is a lower bound for \(\set{f(x)}{a \leq x \leq b}\text{,}\) it follows that \(m \leq m_1\text{,}\) hence
    \begin{equation*} m(b-a) \leq m_1(b-a) = L(f,P_1) \leq \Lint{a}{b}f = \int_a^b f\text{.} \end{equation*}
    On the other hand, since \(M\) is an upper bound for \(\set{f(x)}{a \leq x \leq b}\text{,}\) it follows that \(M_1 \leq M\text{,}\) hence
    \begin{equation*} \int_a^b f = \Uint{a}{b}f \leq L(f,P_1) = M_1(b-a) \leq M(b-a)\text{.} \end{equation*}

    Proof of Theorem 1.6.1.

    Before continuing, let us note that \(F\) is well-defined. Since \(F(a) \defn \int_a^a f \defn 0\) and \(F(b) \defn \int_a^b\text{,}\) it suffices to check that \(F(x)\) is well defined when \(a \lt x \lt b\text{.}\) Indeed, let such \(x\) be given. Then, by Proposition 1.4.11, the function \(f\) is integrable on both \([a,x]\) and \([x,b]\text{,}\) so that \(F(x) \defn \int_a^x f\) is well-defined.
    Now, since the integrable function \(f\) must be bounded on \([a,b]\text{,}\) choose \(M \gt 0\) such that \(\abs{f(x)} \leq M\) for all \(a \leq x \leq b\text{.}\) We will find it useful to show that \(\abs{F(t)-F(s)} \leq M(t-s)\) for all \(s\) and \(t\) satsfiying \(a \leq s \lt t \leq b\text{.}\) Indeed, let such \(s\) and \(t\) be given. Then, by definition of \(F\) and Proposition 1.4.11,
    \begin{equation*} F(t) = \int_a^t f = \int_a^s f + \int_s^t f = F(s) + \int_s^t f\text{,} \end{equation*}
    so that \(F(t)-F(s) = \int_s^t f\text{.}\) But now, since \(-M \leq f(x) \leq M\) for all \(s \leq x \leq t\text{,}\) it follows by Lemma 1.6.3 that \(-M(t-s) \leq \int_a^b f \leq M(t-s)\text{.}\) Hence,
    \begin{equation*} \abs{F(t)-F(s)} = \abs{\int_a^b f} = \abs{M(t-s)} = M(t-s)\text{.} \end{equation*}
    At last, let us show that \(F\) is continuous. Let \(a \leq c \leq b\) be given; we’ll show that \(F\) is continuous at \(c\text{.}\) Thus, let \(\epsilon \gt 0\) and set \(\delta \defn \frac{\epsilon}{M}\text{.}\) Let \(a \leq x \leq b\) satisfy \(\abs{x-c} \lt \delta\text{.}\)
    1. If \(x \gt c\text{,}\) then
      \begin{equation*} \abs{F(x)-F(c)} \leq M(x-c) \lt M\delta = M\cdot\frac{\epsilon}{M} = \epsilon\text{.} \end{equation*}
    2. If \(x = c\text{,}\) then \(\abs{F(x)-F(c)} = 0 \lt \epsilon\text{.}\)
    3. If \(x \lt c\text{,}\) then
      \begin{equation*} \abs{F(x)-F(c)} = \abs{F(c)-F(x)} \leq M(c-x) \lt M\delta = M\cdot\frac{\epsilon}{M} = \epsilon\text{.} \end{equation*}
    Thus, in any case, we obtain \(\abs{F(x)-F(c)} \lt \epsilon\text{.}\)
    We now state and prove the first fundamental theorem of calculus, which guarantees that the function \(F\) of Theorem 1.6.1 is differentiable with derivative \(F^\prime = f\) whenever \(f\) is continuous.

    Theorem 1.6.4. First fundamental theorem of calculus.

    Let \(f\) be an integrable function on a closed interval \([a,b]\text{.}\) Let \(F\) be the function \([a,b]\) given by \(F(x) \defn \int_a^x f\text{.}\) Let \(a \leq c \leq b\text{.}\) If \(f\) is continuous at \(c\text{,}\) then \(F\) is differentiable at \(c\) and \(F^\prime(c) =f(c)\text{.}\)

    Remark 1.6.5.

    Crudely speaking, results recognizable as the first fundamental theorem of calculus can be found in the work of James Gregory

    and Isaac Barrow

    in the mid 17th century. However, it was Gottfried Wilhelm Leibniz
     4

    and Sir Isaac Newton
     5 
    en.wikipedia.org/wiki/Isaac_Newton
    who understood its true significance and placed it in the theoretical context we use today.

    Proof of Theorem 1.6.4.

    For simplicity, let us suppose that \(f\) is continuous at \(c\) satisfying \(a \lt c \lt b\text{.}\) By definition of the derivative
     6 
    opentext.uleth.ca/apex-calculus/sec_derivative.html#def_derivative_at_a_point
    , need to show that \(\lim_{h \to 0}\frac{F(c+h)-F(c)}{h} = f(c)\text{.}\) For non-zero \(h\) satisfying \(a-c \lt h \lt b-c\text{,}\) let
    \begin{align*} m_h \amp \defn \begin{cases}\inf\set{f(x)}{c+h \leq h \leq c} \amp \text{if } h \lt 0\\ \inf\set{f(x)}{c \leq h \leq c+h} \amp \text{if } h \gt 0 \end{cases}\\ M_h \amp \defn \begin{cases}\sup\set{f(x)}{c+h \leq h \leq c} \amp \text{if } h \lt 0\\ \sup\set{f(x)}{c \leq h \leq c +h} \amp \text{if } h \gt 0 \end{cases}\text{.} \end{align*}
    On the one hand, we’ll show that \(m_h \leq \frac{F(c+h)-F(c)}{h} \leq M_h\) for all such \(h\text{.}\) On the other hand, we’ll show that \(\lim_{h\to 0}m_h = f(c) = \lim_{h\to 0}M_h\text{.}\) The squeeze theorem
     7 
    opentext.uleth.ca/apex-calculus/sec_limit_analytically.html#thm_sqz
    will then do the rest.
    Let us first show that \(m_h \leq \frac{F(c+h)-F(c)}{h} \leq M_h\) for all non-zero \(h\) such that \(a-c \lt h \lt b-c\text{.}\) For simplicity, let \(0 \lt h \lt b-c\) be given (the complementary case where \(a-c \lt h \lt 0\) will follow similarly). On the one hand, by the proof of Theorem 1.6.1, we know that \(F(c+h)-F(c) = \int_c^{c+h}f\text{.}\) On the other hand, since \(m_h \leq f(x) \leq M_h\) for all \(c \leq x \leq c+h\text{,}\) it therefore follows by Lemma 1.6.3 that
    \begin{equation*} m_h h = m_h((c+h)-c) \leq \int_c^{c+h} f \leq M_h((c+h)-c) = M_h h\text{.} \end{equation*}
    Thus, we find that \(m_h h \leq F(c+h)-F(c) \leq M_h h\text{,}\) which implies our desired inequality since \(h \gt 0\text{.}\)
    Let us now show that \(\lim_{h \to 0}m_h = c\) and \(\lim_{h \to 0}M_h = c\text{;}\) we’ll do so in tandem. Let \(\epsilon \gt 0\) be given. Since \(f\) is continuous at \(c\text{,}\) choose \(\delta \gt 0\) such that \(\abs{f(x)-f(c)} \lt \epsilon\) whenever \(\abs{x-c} \lt \delta\text{.}\) Now, let \(a-c \lt h \lt b-c\) satisfy \(0 \lt \abs{h} \lt \delta\text{.}\) Then \(f(c) - \epsilon \lt f(x) \lt f(c) + \epsilon\) for all \(x\) satisfying \(\abs{x-c} \leq \abs{h}\text{,}\) so that
    \begin{equation*} f(c) - \epsilon \lt m_h \leq M_h \lt f(c) + \epsilon \end{equation*}
    whether \(h \lt 0\) or \(h \gt 0\text{,}\) so that \(\abs{m_h-f(c)} \lt \epsilon\) and \(\abs{M_h-f(c)} \lt \epsilon\text{.}\)
    Remarkably, one can adapt the proof of Theorem 1.6.4 to yield a proof that a continuous function on a closed interval \([a,b]\) is necessarily integrable.

    Theorem 1.6.6.

    Let \(f\) be a continuous function on a closed interval \([a,b]\text{.}\) Then \(f\) is integrable on \([a,b]\text{.}\)

    Sketch of proof of Theorem 1.6.6.

    Define functions \(F_L\) and \(F_U\) on \([a,b]\) by \(F_L(x) \defn \Lint{a}{x}f\) and \(F_U(x) \defn \Uint{a}{x}f\text{,}\) respectively, with the convention that \(\Lint{a}{a}f = 0 \Uint{a}{a}f\) Using Proposition 1.3.10, one can adapt the proof of Theorem 1.6.4 to show that both \(F_L\) and \(F_U\) are differentiable on \([a,b]\) and \(F_L^\prime = f = F_U^\prime\text{.}\) Hence, \((F_U-F_L)^\prime = F_U^\prime - F_L^\prime = 0\) on \([a,b]\text{,}\) it follows
     8 
    opentext.uleth.ca/apex-calculus/sec_mvt.html#thm-mvt-corollaries-2-1
    that \(F_U-F_L\) is constant, so that
    \begin{equation*} F_U(x)-F_L(x) = F_U(a) - F_L(a) = 0 - 0 = 0 \end{equation*}
    for all \(a \leq x \leq b\text{.}\)
