    Section 1.2 Upper Bounds and Lower Bounds

    In Section 1.1, we derived the area of a planar region by obtaining better and better upper and lower estimates for its true value. In this section, we make precise the notion of estimating a number or set of numbers from above (with an upper bound) or below (with a lower bound). This, in turn, leads us to consider the notion of least upper bound (optimal upper bound) and greatest lower bound (optimal lower bound), which generalize maximal and minimal element, respectively, of a finite set of numbers.
    The existence of least upper bounds and greatest lower bounds is a fundamental property of the real numbers that was first identified by Bernard Bolzano
     1 
    en.wikipedia.org/wiki/Bernard_Bolzano
    in the early 19th century. It turns out to be the property of real numbers that distinguishes them from the rational numbers—for instance, it’s the reason why \(2\) has a real square root. In differential calculus, it gives rise to the intermediate value theorem
     2 
    opentext.uleth.ca/apex-calculus/sec_continuity.html#thm_IVT
    and extreme value theorem
     3 
    opentext.uleth.ca/apex-calculus/sec_extreme_values.html#thm_extreme_val
    . In this course, it will give rise to the rigorous theory of integration.

    Subsection 1.2.1 Upper bounds

    Consider the set \((0,1] = \set{x \in \R}{0 \lt x \leq 1}\text{.}\) By definition, \(x \leq 1\) for every \(x \in (0,1]\text{;}\) thus, \(1\) is a simultaneous upper estimate for all numbers in the set \((0,1]\text{.}\) Likewise, \(x \leq 2025\) for every \(x \in (0,1]\)); thus, \(2025\) is also a simultaneous upper estimate for all numbers in the set \((0,1]\text{,}\) if a much cruder upper estimate than \(1\text{.}\) The notion of a simultaneous upper estimate for all numbers in some set of numbers is formalized as follows.

    Definition 1.2.1.

    Let \(A\) be a set of real numbers. An upper bound for \(A\) is a real number \(M\) such that \(x \leq M\) for all \(x \in A\text{.}\) Thus, we say that \(A\) is bounded above if there exists an upper bound for \(A\text{.}\)

    Remark 1.2.2.

    Note that \(M\) is an upper bound for \(A\) if and only if \(A \subset (0,M]\text{.}\)

    Example 1.2.3.

    The number \(1\) is an upper bound for \((0,1]\text{,}\) so that \((0,1]\) is bounded above. The number \(2025\) is also an upper bound for \((0,1]\text{.}\)

    Example 1.2.4.

    Let \(A_1\) be the set of all numbers of the form \(\frac{1}{3} - \frac{1}{2n} + \frac{1}{6n^2}\text{,}\) where \(n \in \N\) (look familiar?). Given \(n \in \N\text{,}\) since \(\frac{1}{2n} - \frac{1}{6n^2} = \frac{3n-1}{6n^2} \geq 0\text{,}\) it follows that
    \begin{equation*} \frac{1}{3} - \frac{1}{2n} + \frac{1}{6n^2} = \frac{1}{3} - \frac{3n-1}{6n^2} \leq \frac{1}{3}\text{.} \end{equation*}
    Thus, \(\frac{1}{3}\) is an upper bound for \(A_1\text{.}\)

    Remark 1.2.5.

    To show that a set of real numbers \(A\) is not bounded above, you need to show that every would-be upper bound \(M\) fails to be an upper bound for \(A\text{.}\) More precisely, you need to show that for every \(M \in A\text{,}\) there exists some \(x \in A\) such that \(x \gt M\text{.}\)

    Example 1.2.6.

    The set of all real numbers \(\R\) is not bounded above. Indeed, let \(M \in \R\) be given. Then \(x \defn M+1\) satisfies \(x \in \R\) and \(x > M\text{.}\)
    Note that the upper bound \(1\) for \((0,1]\) gives you more information than the much larger upper bound \(2025\text{.}\) For example, the fact that \(1\) is an upper bound for \((0,1]\) lets you conclude that \(2 \notin (0,1]\) since \(2 \gt 1\text{.}\) By contrast, since \(2 \leq 2025\text{,}\) the larger upper bound \(2025\) is too crude to detect that \(2 \notin (0,1]\text{.}\) Thus, we formalize the notion of an optimal upper bound.

    Definition 1.2.7.

    Let \(A\) be a set of real numbers that is bounded above, and let \(M\) be a real number. We say that \(M\) is the least upper bound or supremum of \(A\) whenever it satisfies both of the following conditions:
    1. \(M\) is an upper bound for \(A\text{;}\)
    2. if \(M'\) is any other upper bound for \(A\text{,}\) then \(M \leq M'\text{.}\)
    Hence, we denote the least upper bound \(M\) of \(A\) by \(\sup A\text{.}\)

    Fact 1.2.8.

    Let \(A\) be a set of real numbers. If \(A\) has a least upper bound, then it is unique, so that the notation \(\sup A\) is unambiguous.

    Proof.

    Suppose that \(M_1\) and \(M_2\) are both least upper bounds of a set of real numbers \(A\text{.}\) On the one hand, since \(M_1\) is a least upper bound and \(M_2\) is an upper bound, it follows that \(M_1 \leq M_2\text{.}\) On the other hand, since \(M_2\) is a least upper bound and \(M_1\) is an upper bound, it follows that \(M_2 \leq M_1\text{.}\) Thus, it follows that \(M_1 = M_2\text{.}\)
    There’s a nice special case where you can find the least upper bound of a set without any trouble. This case encompasses, for instance, the set \((0,1]\text{.}\)

    Proposition 1.2.9.

    Let \(A\) be a set of real numbers, and suppose that \(A\) has a maximal element \(\max A\text{,}\) i.e., \(\max A \in A\) and \(x \leq \max A\) for all \(x \in A\text{.}\) Then \(\sup A = \max A\text{.}\)

    Proof.

    First, note that the maximal element \(\max A\) of \(A\) is, in particular, an upper bound for \(A\text{.}\) Now, suppose that \(M\) is any other upper bound for \(A\) Then, since \(\max A \in A\text{,}\) it follows that \(\max A \leq M\text{.}\) Thus, \(\max A\) is the least upper bound of \(A\text{.}\)

    Example 1.2.10.

    The maximal element \(1\) of \((0,1]\) is, therefore, the least upper bound of \((0,1]\text{,}\) i.e., \(\sup(0,1] = 1\text{.}\)
    We can now state the fundamental property of real numbers that distinguished the field of real numbers \(\R\) from the field of rational numbers \(\Q\text{.}\) Indeed, it will turn out to be the property of real numbers that makes calculus work as a mathematical theory.

    Axiom 1.2.11 (Bolzano). Least-upper-bound property of the real numbers.

    Let \(A\) be a set of real numbers. If \(A\) is non-empty and bounded above, then its least upper bound \(\sup A\) exists.

    Remark 1.2.12.

    Every real number \(M\) is vacuously an upper bound for the empty set \(\emptyset\text{:}\) the condition that \(x \leq M\) for all \(x \in \emptyset\) is satisfied precisely because the empty set has no elements.

    Theorem 1.2.13.

    The set of natural numbers \(\N\) is not bounded above.

    Proof.

    We’ll prove this by contradiction. Assume for the sake of argument that \(\N\) is bounded above. Then \(\N\) is non-empty (e.g., since \(1 \in \N\)) and bounded above (by assumption), so that \(\N\) has a least upper bound \(\sup \N\)by Axiom 1.2.11.
    Now, let \(m \in \N\) be given. Since \(m+1 \in \N\text{,}\) it follows that \(m+1 \leq \sup\N\text{,}\) hence \(m \leq \sup\N - 1\text{.}\) Since \(m \in \N\) was arbitrary, this shows that \(M \defn \sup N - 1\) is also an upper bound. Thus, \(M\) is an upper bound of \(\N\) that satisfying \(M \lt \sup N\text{,}\) which contradicts the definition of \(\sup N\text{.}\)

    Corollary 1.2.14. Axiom 1.1.14.

    For every positive real number \(\epsilon\text{,}\) there exists a natural number \(N\) such that \(\frac{1}{N} \lt \epsilon\text{.}\)

    Proof.

    By Theorem 1.2.13, the set \(\N\) is not bounded above, so \(\frac{1}{\epsilon}\) cannot be an upper bound of \(\N\text{.}\) Thus, there exists \(N \in \N\) such that \(\frac{1}{\epsilon} \lt N\text{,}\) which implies, in turn, that \(\epsilon \gt \frac{1}{N}\text{.}\)

    Example 1.2.15.

    We can now show that \(\frac{1}{3}\) is the least upper bound of the set \(A_1\) of Example 1.2.4. Let \(M\) be a real number, and suppose that \(M \gt \frac{1}{3}\text{;}\) we’ll show that \(M\) is not an upper bound for \(A_1\) by finding \(x \in A_1\) such that \(x \gt M\text{.}\)
    Since \(M \gt \frac{1}{3}\) by assumption, it follows that \(M - \frac{1}{3} \gt 0\text{.}\) Thus, by Corollary 1.2.14, there exists \(N \in \N\) such that \(\frac{1}{N} \lt M-\frac{1}{3}\text{.}\) Then \(x := \frac{1}{3} - \frac{1}{2N} + \frac{1}{6N^2} \in A_1\) satisfies
    \begin{align*} x \amp = \frac{1}{3} - \frac{1}{2N} + \frac{1}{6N^2}\\ \amp \geq \frac{1}{3} - \frac{1}{N}\\ \amp \gt \frac{1}{3} - \left(M-\frac{1}{3}\right)\\ \amp = M\text{.} \end{align*}

    Remark 1.2.16.

    Note that \(\frac{1}{3} = \sup A_1\) but \(\frac{1}{3} \notin A_1\) since \(\frac{1}{3}-\frac{1}{2n}+\frac{1}{6n^2} \lt \frac{1}{3}\) for all \(n \in \N\text{.}\) Compare this to the case of \(1 = \sup(0,1] = \max(0,1]\text{.}\)

    Subsection 1.2.2 Lower bounds

    Again, consider the set \((0,1]\text{.}\) By definition, \(0 \leq x\) for every \(x \in (0,1]\text{;}\) thus, \(0\) is a simultaneous lower estimate for all numbers in the set \((0,1]\text{.}\) Likewise, \(-2025 \leq x\) for every \(x \in (0,1]\)); thus, \(-2025\) is also a simultaneous lower estimate for all numbers in the set \((0,1]\text{.}\) The notion of a simultaneous lower estimate for all numbers in some set of numbers is formalized as follows.

    Definition 1.2.17.

    Let \(A\) be a set of real numbers. A lower bound for \(A\) is a real number \(m\) such that \(m \leq x\) for all \(x \in A\text{.}\) Thus, we say that \(A\) is bounded below if there exists a lower bound for \(A\text{.}\)

    Remark 1.2.18.

    Note that \(m\) is a lower bound for \(A\) if and only if \(A \subset [m,\infty)\text{.}\)

    Example 1.2.19.

    The number \(0\) is a lower bound for \((0,1]\text{,}\) so that \((0,1]\) is bounded blow. The number \(-2025\) is also a lower bound for \((0,1]\text{.}\)

    Example 1.2.20.

    Let \(A_2\) be the set of all numbers of the form \(\frac{1}{3} + \frac{1}{2n} + \frac{1}{6n^2}\text{,}\) where \(n \in \N\) (look familiar?). Given \(n \in \N\text{,}\) since \(\frac{1}{2n} + \frac{1}{6n^2} \geq 0\text{,}\) it follows that
    \begin{equation*} \frac{1}{3} + \frac{1}{2n} + \frac{1}{6n^2} \geq \frac{1}{3}\text{.} \end{equation*}
    Thus, \(\frac{1}{3}\) is a lower bound for \(A_1\text{.}\)

    Remark 1.2.21.

    To show that a set of real numbers \(A\) is not bounded below, you need to show that every would-be lower bound \(M\) fails to be an upper bound for \(A\text{.}\) More precisely, you need to show that for every \(M \in A\text{,}\) there exists some \(x \in A\) such that \(x \lt M\text{.}\)

    Example 1.2.22.

    The set of all real numbers \(\R\) is not bounded below. Indeed, let \(m \in \R\) be given. Then \(x \defn m-1\) satisfies \(x \in \R\) and \(x \lt m\text{.}\)
    Much like before, we can formalize a notion of optimal lower bound.

    Definition 1.2.23.

    Let \(A\) be a set of real numbers that is bounded below, and let \(m\) be a real number. We say that \(m\) is the greatest upper bound or infimum of \(A\) whenever it satisfies both of the following conditions:
    1. \(m\) is an lower bound for \(A\text{;}\)
    2. if \(m'\) is any other lower bound for \(A\text{,}\) then \(m \geq m'\text{.}\)
    Hence, we denote the greatest lower bound \(M\) of \(A\) by \(\sup A\text{.}\)

    Note 1.2.24.

    Uniqueness of greatest lower bounds follows from a qualitatively identical argument to uniqueness of least upper bounds.

    Example 1.2.25.

    The lower bound \(0\) of \((0,1]\) is its greatest lower bound, i.e., \(0 = \inf (0,1]\text{.}\) Indeed, let \(m \gt 0\text{;}\) we’ll show that \(m\) cannot be a lower bound for \((0,1]\) by demonstrating \(x \in (0,1]\) such that \(x \lt m\text{.}\) If \(m \gt 1\text{,}\) then \(x \defn 1 \in (0,1]\) satisfies \(x \lt m\text{.}\) If, instead, \(0 \lt m \leq 1\text{,}\) then \(x \defn \frac{m}{2} \in (0,1]\) satisfies \(x \lt m\text{.}\)
    The most important single example of a greatest lower bound is the following.

    Proposition 1.2.26.

    Let \(S\) be the set of all numbers of the form \(\frac{1}{n}\text{,}\) where \(n \in \N\text{.}\) Then \(\inf S = 0\text{.}\)

    Proof.

    Since \(S\) consists of positive numbers, it follows that \(0\) is a lower bound for \(S\text{.}\) Now, let \(m \gt 0\) be given; we’ll show that \(m\) cannot be a lower bound by demonstrating \(x \in S\) such that \(x \lt m\text{.}\) Indeed, by Corollary 1.2.14, there exists \(N \in \N\) such that \(\frac{1}{N} \lt m\text{.}\) Then \(x \defn \frac{1}{N} \in S\) satisfies \(x \lt \text{.}\)

    Example 1.2.27.

    One can now show that \(\frac{1}{3}\) is the greatest lower bound of the set \(A_2\) of Example 1.2.20. Indeed, one can very easily modify the argument of Example 1.2.15 into a proof identical in structure to that of Proposition 1.2.26.
    Every fact about (least) upper bounds has a corresponding fact about (greatest) lower bounds. This correspondence is provided by the following easy-to-prove fact.

    Proposition 1.2.28.

    Let \(A\) be a non-empty set of real numbers, and let \(-A \defn \set{x \in \R}{-x \in A}\text{.}\)
    1. A real number \(c\) is a lower bound for \(A\) if and only if \(-c\) is an upper bound for \(-A\text{.}\)
    2. The set \(A\) has a greatest lower bound if and only if the set \(-A\) has a least upper bound, in which case \(\inf A = -\sup(-A)\text{.}\)

    Corollary 1.2.29.

    Let \(A\) be a set of real numbers, and suppose that \(A\) has a minimal element \(\min A\text{,}\) i.e., \(\min A \in A\) and \(x \geq \min A\) for all \(x \in A\text{.}\) Then \(\inf A = \min A\text{.}\)

    Example 1.2.30.

    The number \(1\) is the minimal element of \(\N\text{.}\) Thus, it follows that \(\inf\N = \min\N = 1\text{.}\)

    Corollary 1.2.31 (Bolzano). Least-upper-bound property of the real numbers.

    Let \(A\) be a set of real numbers. If \(A\) is non-empty and bounded below, then its greatest lower bound \(\inf A\) exists.
