    Section 1.9 Geometric and Physical Interpretations

    Subsection 1.9.1 Signed area

    Definition 1.9.1.

    Let \(U\) be a set of points in the plane \(\R^2\text{.}\) Suppose that \(U\) admits a description of the form
    \begin{equation*} U = \set{(x,y) \in \R^2}{a \leq x \leq b, \, f(x) \leq y \leq g(x)}\text{,} \end{equation*}
    where \(f\) is integrable and non-negative on \([a,b]\text{.}\) The area of \(U\) is
    \begin{equation*} \Area(U) \defn \int_a^b f\text{.} \end{equation*}

    Remark 1.9.2.

    This definition recovers the working definition of area used in Section 1.1. In particular, it turns to be consistent with the rigorous notion of Jordan content discussed in Assumption 1.1.4.

    Example 1.9.3.

    Let \(U\) be the region in the plane enclosed by the curves \(y=1-x^4\) and \(y=0\text{.}\) We’ll compute \(\Area(U)\) by following the recipe of Definition 1.9.1.
    A quick sketch (see Figure 1.9.4) shows that
    \begin{equation*} U = \set{(x,y) \in \R^2}{a \leq x \leq b,\, 0 \leq y \leq 1-x^4}\text{,} \end{equation*}
    where \(x=a\) and \(x=b\) are the \(x\)-coordinates of the respective intersection points of the curve \(y=1-x^4\) and \(y=0\text{,}\) i.e., the solutions of \(1-x^4 = 0\text{.}\) Since
    \begin{equation*} 1-x^4 = (1-x^2)(1+x^2) = (1-x)(1+x)(1+x^2)\text{,} \end{equation*}
    it follows that \(1-x^4 = 0\) if and only if \((1-x)(1+x) = 0\text{,}\) if and only if \(x=-1\) or \(x=1\text{.}\) Thus, we obtain
    \begin{equation*} U = \set{(x,y) \in \R^2}{-1 \leq x \leq 1,\, 0 \leq y \leq 1-x^4}\text{,} \end{equation*}
    so that \(\Area(U) = \int_{-1}^{1} (1-x^4) \du{x}\text{.}\)
    Now, by (1.8.2) applied term by term,
    \begin{equation*} \int (1-x^4)\du{x} = \int x^0 \du{x} - \int x^4 \du{x} = x - \frac{x^5}{5} + C. \end{equation*}
    Hence, by Theorem 1.7.3, we conclude that
    \begin{align*} \Area(U) \amp = \int_{-1}^{1} (1-x^4) \du{x} \\ \amp = \eval{x-\frac{x^5}{5}}{-1}{1} \\ \amp = \left(1-\frac{1}{5}\right) - \left(-1+\frac{1}{5}\right)\\ \amp = \frac{8}{5} \text{.} \end{align*}
    But now, does an integral \(\int_a^b f\) admit a geometric interpretation when \(f\) takes negative values? The answer is yes.

    Proposition 1.9.5.

    Let \(f\) be an integrable function on \([a,b]\text{.}\) Define functions \(f_+\) and \(f_-\) on \([a,b]\) by \(f(x) \defn \max(f(x),0)\) and \(f(x) \defn\max(-f(x),0)\text{,}\) respectively. Then \(f_+\) and \(f_-\) are both integrable and non-negative on \([a,b]\) and satisfy \(f = f_+ - f_-\text{,}\) so that
    \begin{equation*} \int_a^b f = \int_a^b f_+ - \int_a^b f_-\text{.} \end{equation*}

    Heuristic 1.9.6.

    If you accept Assumption 1.1.4, that reflections preserve the area of planar regions, and that Definition 1.9.1 is consistent with Assumption 1.1.4, then
    \begin{align*} \int_a^b f_+ \amp = \Area\set{(x,y) \in \R^2}{a \leq x \leq b, \, 0 \leq y \leq f_+(x)} \\ \amp = \Area\set{(x,y) \in \R^2}{a \leq x \leq b, \, 0 \leq y \leq f(x)} \end{align*}
    gives the area above the \(x\)-axis and below the graph of \(f\) on \([a,b]\)
    \begin{align*} \int_a^b f_- \amp = \Area\set{(x,y) \in \R^2}{a \leq x \leq b, \, 0 \leq y \leq f_-(x)} \\ \amp = \Area\set{(x,y) \in \R^2}{a \leq x \leq b, \, f(x) \leq y \leq 0} \end{align*}
    gives the area below the \(x\)-axis and above the graph of \(f\) on \([a,b]\text{.}\)
    For a proof of the non-trivial part of Proposition 1.9.5 (i.e., integrability of \(f_+\) and \(f_-\)), look here

    Subsection 1.9.2 Displacement and distance

    Consider a point mass moving along some straight line from time \(a\) to time \(b\text{.}\) By imposing a Cartesian coordinate system (including unit of length) on the straight line, we can encode the position of our point mass as a function \(q\) of time \(t \in [a,b]\text{,}\) so that \(q(t)\) gives the position at time \(t\text{.}\) Let’s suppose that \(q\) is continuously differentiable, so that \(q^\prime\) gives velocity as a function of time.
    1. By Theorem 1.7.3, the integral \(\int_a^b q^\prime = q(b) - q(a)\) gives the net displacement as a result of the motion.
    2. By contrast, speed as a function of time is given by \(\abs{q'}\text{,}\) so that the integral \(\int_a^b \abs{q^\prime}\) gives the distance travelled as a result of the motion.
    The following example will demonstrate just how different net displacement and distance travelled can be.

    Example 1.9.7.

    Consider a simple harmonic oscillator (e.g. a block on a spring oscillating along a straight track in the absence of friction, resistance, or forcing) starting from rest, so that \(q(t) = \cos(t)\) in suitable units.
    1. The net displacement from time \(0\) to time \(2\pi\) is
      \begin{equation*} \int_0^{2\pi} q^\prime = q(2\pi) - q(0) = \cos(2\pi) - \cos(0) = 0\text{;} \end{equation*}
      this means that you ended up at time \(2\pi\) exactly where you started at time \(0\text{.}\)
    2. Note that \(q^\prime(t) = \cos^\prime(t) = -\sin(t)\text{,}\) which is non-negative for \(0 \leq t \leq \pi\) and non-positive for \(\pi \leq t \leq 2\pi\text{.}\) Thus,
      \begin{equation*} \abs{q^\prime(t)} = \abs{\sin(t)} = \begin{cases} \sin(t) \amp 0 \leq t \leq \pi \\ -\sin(t) \amp \pi \leq t \leq 2\pi \end{cases} \end{equation*}
      so that the total distance travelled is
      \begin{align*} \int_0^{2\pi}\abs{q^\prime(t)}\du{t} \amp = \int_0^{\pi}\abs{q^\prime(t)}\du{t} + \int_{\pi}^{2\pi} \abs{q^\prime(t)}\du{t} \\ \amp = \int_0^{\pi} \sin(t)\du{t} + \int_{\pi}^{2\pi}(-\sin(t))\du{t} \\ \amp = \int_0^{\pi} \sin(t)\du{t} - \int_{\pi}^{2\pi}\sin(t)\du{t} \\ \amp = \eval{-\cos(t)}{0}{\pi} + \eval{\cos(t)}{\pi}{2\pi} \\ \amp = -\cos(\pi) + \cos(0) + \cos(2\pi) - \cos(\pi) \\ \amp = 4\text{.} \end{align*}
