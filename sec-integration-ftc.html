    Section 1.8 Integration via Antidifferentiation

    Let’s recall the second fundamental theorem of calculus (Theorem 1.7.3):
    Let \(f\) be an integrable function on a closed interval \([a,b]\text{.}\) Suppose that \(F\) is an antiderivative of \(f\) on \([a,b]\text{.}\) Then \(\int_a^b f = F(b) - F(a)\text{.}\) Equivalently, in Lebniz notation,
    \begin{equation*} \int_a^b f(x)\du{x} = F(x) \vert_a^b\text{,} \end{equation*}
    where \(F(x) \vert_a^b \defn F(b)-F(a)\text{.}\)
    In principle, this means that we can integrate a function \(f\) by reverse-engineering an antiderivative of \(f\text{.}\) This gives rise to the traditional notion of indefinite integration.

    Convention 1.8.1. Indefinite integration.

    1. Let \(f\) be an integrable function on an interval \(I\) that has an antiderivative. The indefinite integral \(\int f(x) \du{x}\) of \(f\) is a symbolic placeholder for any antiderivative of \(f\text{.}\)
    2. Any equality of functions involving an indefinite integral \(\int f(x) \du{x}\) holds, by definition, up to addition of a constant term, which is typically denoted by \(C\text{.}\)

    Note 1.8.2.

    A constant is a real number \(C\) treated as a constant function with constant value \(C\text{.}\)

    Remark 1.8.3.

    If you want to be rigorous, you can define the indefinite integral of a function \(f\) on an interval admitting antiderivatives to be the set \(\int f\) of all anti-derivatives of \(f\text{.}\) Then an equality of indefinite integrals can be made precise as an equality of functions modulo constant functions. Compare angles, where two numbers \(\theta_1\) and \(\theta_2\) represent the same angle in radians whenever \(\theta_1-\theta_2 = 2\pi k\) for some integer \(k\text{.}\)
    Every fact about derivatives yields a corresponding fact about indefinite integrals. The most basic facts about derivatives immediately yield the following basic facts about indefinite integrals.

    Proposition 1.8.4. Basic properties of indefinite integrals.

    1. Let \(f\) be an integrable function on an interval \(I\text{.}\) Suppose that \(F\) is an antiderivative of \(f\text{.}\) Then
      \begin{equation*} \int f(x) \du{x} = F(x) + C\text{.} \end{equation*}
      This means that a antiderivative of \(f\) is precisely a function of the form \(F + C\text{,}\) where \(C\) is a constant.
    2. Let \(f\) and \(g\) be integrable functions on an interval \(I\) that have antiderivatives. Then
      \begin{equation*} \int \left(f(x)+g(x)\right)\du{x} = \int f(x) \du{x} + \int g(x) \du{x}\text{.} \end{equation*}
      This means that an antiderivative of \(f+g\) is precisely a function of the form \(F+G+C\text{,}\) where \(F\) is an antiderivative of \(f\text{,}\) \(G\) is an antiderivative of \(g\text{,}\) and \(C\) is a constant.
    3. Let \(f\) be an integrable function on an interval \(I\) that has an antiderivative, and let \(\lambda\) be a real number. Then
      \begin{equation} \int \lambda f(x) \du{x} = \lambda \int f(x) \du{x}\text{.}\tag{1.8.1} \end{equation}
      This means that an antiderivative of \(\lambda f\) is precisely a function of the form \(\lambda F + C\text{,}\) where \(F\) is an antiderivative of \(f\) and \(C\) is a constant.

    Warning 1.8.5.

    Let \(f\) be an integrable function on an interval \(I\) that has an antiderivative. The (1.8.1) does not imply that \(\int 0 f(x) \du{x} = 0\text{.}\) Instead, it says that \(\int 0 f(x) \du{x} = 0\) up to addition of a constant term, i.e., the antiderivatives of \(0f = 0\) are precisely the constants.

    Example 1.8.6.

    Let \(N\) be a non-negative integer. By the proof of Example 1.7.5, the function \(F(x) \defn \frac{x^{N+1}}{N+1}\) is an antiderivative of the function \(f(x) \defn x^N\text{.}\) Hence,
    \begin{equation} \int x^N \du{x} = \frac{x^{N+1}}{N+1} + C\text{.}\tag{1.8.2} \end{equation}
    Let’s apply this to computing \(\int_0^2 (x^3+2x-1)\du{x}\text{.}\) By repeated applications of Proposition 1.8.4 and (1.8.2), it follows that
    \begin{align*} \int (x^3+2x-1) \du{x} \amp = \int (x^3+2x-1) \du{x} \\ \amp = \int x^3 \du{x} + 2 \int x \du{x} - \int 1 \du{x} \\ \amp = \frac{x^4}{4} + 2 \frac{x^2}{2} - x + C \\ \amp = \frac{x^4}{4} + x^2 - x + C \text{,} \end{align*}
    where we used our convention on equalities of involving indefinite integrals to “collect constant terms.” Hence, by Theorem 1.7.3, it follows that
    \begin{align*} \int_0^2 (x^3+2x-1) \du{x} \amp = \frac{x^4}{4} + x^2 - x \vert_0^2 \\ \amp = \left(\frac{2^4}{4} + 2^2 - 2\right) - \left(\frac{0^4}{4} + 0^2 - 0\right) \\ \amp = 6 \text{,} \end{align*}
    where we were free to set \(C = 0\) since any choice of antiderivative must yield the same value for \(\int_0^2 (x^3+2x-1) \du{x}\text{.}\)

    Example 1.8.7.

    The natural logarithm \(\log\) is differentiable on \((0,\infty)\) with derivative \(\log^\prime(x) = \frac{1}{x}\text{.}\) Hence, on the domain \((0,\infty)\text{,}\)
    \begin{equation*} \int \frac{1}{x} \du{x} = \log(x) + C\text{.} \end{equation*}
    Let’s apply this to computing \(\int_3^6 \frac{1}{x}\du{x}\text{:}\)
    \begin{equation*} \int_3^6 \frac{1}{x} \du{x} = \log(x) \vert_3^6 = \log(6) - \log(3) = \log(6/3) = \log(2)\text{.} \end{equation*}

    Example 1.8.8.

    Let \(F(x) \defn \log(-x)\) on \((-\infty,0)\text{.}\) Then, by the chain rule, \(F^\prime(x) = \frac{1}{x}\text{.}\) Hence, on the domain \((-\infty,0)\text{,}\)
    \begin{equation*} \int \frac{1}{x} \du{x} = \log(-x) + C\text{.} \end{equation*}

    Example 1.8.9.

    Since the exponential function \(\exp(x) \defn \eu^x\) satisfies \(\exp^\prime = \exp\text{,}\) it follows that
    \begin{equation*} \int \eu^x \du{x} = \eu^x + C\text{.} \end{equation*}
    Thus, for example,
    \begin{equation*} \int_1^3 \eu^x \du{x} = \eu^x \vert_1^3 = \eu^3 - \eu^1 = \eu^3 - \eu\text{.} \end{equation*}

    Example 1.8.10.

    Let’s compute \(\int_0^2 \left(2x^3-6x+\frac{3}{x^2+1}\right)\du{x}\text{.}\) Since
    \begin{align*} \int \left(2x^3-6x+\frac{3}{x^2+1}\right)\du{x} \amp = 2 \int x^3 \du{x} - 6 \int x \du{x} + 3 \int \frac{1}{x^2+1} \du{x} \\ \amp = 2 \cdot \frac{x^4}{4} - 6 \cdot \frac{x^2}{2} + 3\arctan(x) + C\\ \amp = \frac{x^4}{2} - 3x^2 + 3\arctan(x) + C\text{,} \end{align*}
    it follows that
    \begin{align*} \int_0^2 \left(2x^3-6x+\frac{3}{x^2+1}\right)\du{x} \amp = \frac{x^4}{2} - 3x^2 + 3\arctan(x)\vert_0^2 \\ \amp = \left(\frac{2^4}{2} - 3 \cdot 2^2 + 3\arctan(2)\right) - \left(\frac{0^4}{2} - 3 \cdot 0^2 + 3\arctan(0)\right) \\ \amp = -4 + 3\arctan(2) \text{.} \end{align*}

    Example 1.8.11.

    Let’s compute \(\int_1^9 \frac{2t^2+t^2\sqrt{t}-1}{t^2}\du{t}\) in one go:
    \begin{align*} \int_1^9 \frac{2t^2+t^2\sqrt{t}-1}{t^2}\du{t} \amp = \int_1^9 \left(2 + t^{1/2}- t^{-2}\right)\du{t} \\ \amp = 2t + \frac{1}{\frac{1}{2}+1} t^{\frac{1}{2}+1} - \frac{1}{-2}t^{-2+1} \vert_1^9\\ \amp = 2t + \frac{2}{3}t^{3/2} + \frac{1}{2t} \vert_1^9 \\ \amp = \left(2 \cdot 9 + \frac{2}{3}9^{3/2} + \frac{1}{2 \cdot 9}\right) - \left(2 \cdot 1 + \frac{2}{3}1^{3/2} + \frac{1}{2 \cdot 1}\right) \\ \amp = \frac{292}{9}\text{.} \end{align*}
