    Section 1.3 Upper and Lower Integrals

    Let’s revisit the proof of Theorem 1.1.1, where we showed that the planar region
    \begin{equation*} U \defn \set{(x,y) \in \R^2}{0 \leq x \leq 1,\,0 \leq y \leq x^2} \end{equation*}
    must have area \(\frac{1}{3}\text{.}\) Using the machinery of Section 1.2, we can now recast our proof into a robustly generalizable form.
    1. First, in Lemma 1.1.7, we systematically approximated \(U\) from within and without by simpler regions contructed from rectangles, which let us prove the area estimates of Lemma 1.1.9, i.e.,
      \begin{equation*} \frac{1}{3} - \frac{1}{2N} + \frac{1}{6N^2} \leq \Area(U) \leq \frac{1}{3} - \frac{1}{2N} + \frac{1}{6N^2} \end{equation*}
      for every \(N \in \N\text{.}\)
    2. Next, observe that \(\Area(U)\) is both an upper bound for the set
      \begin{equation*} A_1 \defn \set{\frac{1}{3}-\frac{1}{2N}+\frac{1}{6N^2}}{N \in \N} \end{equation*}
      of Example 1.2.4 and a lower bound for the set
      \begin{equation*} A_2 \defn \set{\frac{1}{3}+\frac{1}{2N}+\frac{1}{6N^2}}{N \in \N} \end{equation*}
      of Example 1.2.20. Hence, it follows that
      \begin{equation*} \sup A_1 \leq \Area(U) \leq \inf A_2\text{.} \end{equation*}
    3. Finally, since \(\sup A_1 = \frac{1}{3} = \inf A_2\) by Example 1.2.15 and Example 1.2.27, respectively, it follows that
      \begin{equation*} \frac{1}{3} = \sup A_1 \leq \Area(U) \leq \inf A_2 = \frac{1}{3}\text{.} \end{equation*}
      Hence, at last, \(\Area(U) = \frac{1}{3}\text{.}\)
    In this section, we’ll generalize this process.

    Definition 1.3.1.

    Let \(a \lt b\text{.}\) A partition of the closed interval \([a,b]\) is a finite list of real numbers \(P = (x_0,x_1,\dotsc,x_N)\) such that
    \begin{equation*} a = x_0 \lt x_1 \lt \cdots \lt x_{N-1} \lt x_N = b\text{.} \end{equation*}
    We view a partition \(P = (x_0,x_1,\dotsc,x_N)\) of an interval \([a,b]\) as subdividing \([a,b]\) into \(N\) subintervals \([x_0,x_1],[x_1,x_2],\dotsc,[x_{N-1},x_N]\text{.}\)

    Example 1.3.2.

    Let \(a \lt b\text{,}\) and let \(N \in \N\text{.}\) Define a partition \(P_N = (x_0,x_1,\dotsc,x_N)\) of \([a,b]\) by setting \(x_k \defn a + k \cdot \frac{b-a}{N}\) for all \(0 \leq k \leq N\text{.}\) Then \(P_N\) subdivides \([a,b]\) into \(N\) subintervals \([x_{k-1},x_{k}]\) of equal width \(\frac{b-a}{N}\text{.}\)

    Remark 1.3.4.

    We’ll repeatedly use the fact that for any partition \(P=(x_0,x_1,\dotsc,x_N)\) of \([a,b]\text{,}\) we have \(\sum_{k=1}^N(x_k-x_{k-1}) = b-a\text{.}\) This is called a telescoping sum.

    Definition 1.3.5.

    Let \(f\) be a real-valued function defined on a closed interval \([a,b]\text{.}\) We say that \(f\) is bounded on \(D\) if there exist constants \(m\) and \(M\) such that \(m \leq f(x) \leq M\) for all \(a \leq x \leq b\text{.}\)

    Definition 1.3.6.

    Let \(f\) be a bounded real-valued function on a closed interval \([a,b]\text{,}\) let \(P = (x_0,x_1,\dotsc,x_N)\) be a partition of \([a,b]\text{,}\) and set
    \begin{align*} m_k \amp \defn \inf\set{f(x)}{x_{k-1} \leq x \leq x_k} \amp \\ M_k \amp \defn \sup\set{f(x)}{x_{k-1} \leq x \leq x_k} \end{align*}
    for each \(1 \leq k \leq N\text{.}\)
    1. The lower sum of \(f\) with respect to \(P\) is
      \begin{equation*} L(f,P) \defn \sum_{k=1}^N m_k(x_k-x_{k_1})\text{.} \end{equation*}
    2. The upper sum of \(f\) with respect to \(P\) is
      \begin{equation*} U(f,P) \defn \sum_{k=1}^N M_k(x_k-x_{k_1})\text{.} \end{equation*}

    Observation 1.3.7.

    Let \(f\) be a bounded real-valued function on a closed interval \([a,b]\) such that \(f(x) \geq 0\) for all \(a \leq x \leq \text{.}\) Assume that its undergraph
    \begin{equation*} U_f \defn \set{(x,y) \in \R^2}{a \leq x \leq b, \, 0 \leq y \leq f(x)} \end{equation*}
    has well-defined area (see Assumption 1.1.4). Then
    \begin{equation*} L(f,P) \leq \Area(U_f) \leq U(f,P) \end{equation*}
    for every partition \(P\) of \([a,b]\text{.}\)

    Example 1.3.8.

    Let \(f\) be the function on \([1,2]\) given by \(f(x) \defn \frac{1}{x^2}\text{.}\) Consider the partition \(P_2 = (x_0,x_1,x_2)\) of \([1,2]\) defined by Example 1.3.2, so that \(x_0 = 1\text{,}\) \(x_1 = 1 + 1\cdot \frac{2-1}{2} = \frac{3}{2}\text{,}\) and \(x_2 = 2\text{.}\) Since \(f\) is decreasing on \([1,2]\text{,}\) it follows that
    \begin{equation*} m_k \defn \inf\set{\frac{1}{x^2}}{x_{k-1} \leq x \leq x_k} = \min\set{\frac{1}{x^2}}{x_{k-1} \leq x \leq x_k} = \frac{1}{x_{k}^2} \end{equation*}
    and
    \begin{equation*} M_k \defn \sup\set{\frac{1}{x^2}}{x_{k-1} \leq x \leq x_k} = \max\set{\frac{1}{x^2}}{x_{k-1} \leq x \leq x_k} = \frac{1}{x_{k-1}^2} \end{equation*}
    for \(k=0,1,2\text{.}\) Hence,
    \begin{align*} L(f,P_2) \amp \defn \sum_{k=1}^2 m_k (x_k-x_{k-1})\\ \amp = \frac{1}{x_1^2}(x_1-x_0) + \frac{1}{x_2^2}(x_2-x_1) \\ \amp = \frac{1}{(3/2)^2}(\tfrac{3}{2}-1) + \frac{1}{2^2}(2-\tfrac{3}{2})\\ \amp = \frac{4}{9} \cdot \frac{1}{2} + \frac{1}{4} \cdot \frac{1}{2}\\ \amp =\frac{25}{72} \end{align*}
    and
    \begin{align*} U(f,P_2) \amp \defn \sum_{k=1}^2 M_k (x_k-x_{k-1})\\ \amp = \frac{1}{x_0^2}(x_1-x_0) + \frac{1}{x_1^2}(x_2-x_1) \\ \amp = \frac{1}{1^2}(\tfrac{3}{2}-1) + \frac{1}{(3/2)^2}(2-\tfrac{3}{2})\\ \amp = 1 \cdot \frac{1}{2} + \frac{4}{9} \cdot \frac{1}{2}\\ \amp =\frac{13}{18}\text{.} \end{align*}
    Thus, by Observation 1.3.7, the area of the undergraph \(U_f\) (assuming it’s well-defined) can be estimated from below and above by
    \begin{equation*} \frac{25}{72} \leq \Area(U_f) \leq \frac{13}{18}\text{,} \end{equation*}
    where \(\frac{25}{72} \approx 0.347\) and \(\frac{13}{18}\approx 0.722\text{.}\)

    Example 1.3.9.

    Let \(f\) be the function on \([0,1]\) given by \(f(x) \defn x^2\text{.}\) Let \(N \in \N\) be given, and let \(P_N = (x_0,x_1,\dotsc,x_N)\) be the partition of \([0,1]\) defined by Example 1.3.2, so that \(x_k = 0 + k \cdot \frac{1-0}{N} = \frac{k}{N}\) for all \(0 \leq k \leq N\text{.}\) Since \(f\) is strictly increasing on \([0,1]\text{,}\)
    \begin{align*} m_k \amp \defn \inf\set{x^2}{\frac{k-1}{N} \leq x \leq \frac{k}{N}} \\ \amp = \min\set{x^2}{\frac{k-1}{N} \leq x \leq \frac{k}{N}}\\ \amp = \frac{(k-1)^2}{N^2} \end{align*}
    and
    \begin{align*} M_k \amp \defn \sup\set{x^2}{\frac{k-1}{N} \leq x \leq \frac{k}{N}} \\ \amp = \max\set{x^2}{\frac{k-1}{N} \leq x \leq \frac{k}{N}}\\ \amp = \frac{k^2}{N^2} \end{align*}
    for all \(1 \leq k \leq N\text{.}\) Thus, by the proof of Lemma 1.1.13, it follows that
    \begin{equation*} L(f,P_N) \defn \sum_{k=1}^N m_k(x_k-x_{k-1}) = \sum_{k=1}^N \frac{(k-1)^2}{N^2} \frac{1}{N} = \frac{1}{3} - \frac{1}{2N} + \frac{1}{6N^2} \end{equation*}
    and
    \begin{equation*} U(f,P_N) \defn \sum_{k=1}^N M_k(x_k-x_{k-1}) = \sum_{k=1}^N \frac{k^2}{N^2} \frac{1}{N} = \frac{1}{3} + \frac{1}{2N} + \frac{1}{6N^2}\text{.} \end{equation*}
    By combining these calculations with Observation 1.3.7, we recover the estimates of Lemma 1.1.13.

    Proposition 1.3.10.

    Let \(f\) be a bounded real-valued function on a closed interval \([a,b]\text{.}\)
    1. Let \(P\) be a partition of \([a,b]\text{.}\) Then \(L(f,P) \leq U(f,P)\text{.}\)
    2. Let \(P\) and \(P'\) be partitions of \([a,b]\text{.}\) Suppose that \(P'\) can be obtained from \(P\) by including additional points (i.e., \(P'\) is a refinement of \(P\)). Then
      \begin{equation*} L(f,P) \leq L(f,P') \leq U(f,P') \leq U(f,P)\text{.} \end{equation*}
    3. Let \(P\) and \(P'\) be partitions of \([a,b]\text{.}\) Then \(L(f,P) \leq U(f,P')\text{.}\)

    Proof.

    Let us first prove Item 1 Let \(P = (x_0,x_1,\dotsc,x_N)\) be a partition of \([a,b]\text{.}\) Given \(1 \leq k \leq N\text{,}\) since
    \begin{equation*} m_k = \inf\set{f(x)}{x_{k-1} \leq x \leq x_k} \leq f(x_{k-1}) \leq \sup\set{f(x)}{x_{k-1}\leq x\leq x_k} = M_k\text{,} \end{equation*}
    it follows that \(m_k \leq M_k\text{.}\) Thus,
    \begin{equation*} L(f,P) = \sum_{k=1}^N m_k (x_k-x_{k-1}) \leq \sum_{k=1}^N M_k(x_k-x_{k-1}) = U(f,P)\text{.} \end{equation*}
    Next, let us sketch the proof of Item 2. The general case is a tedious application of the special case where \(P = (a,b)\) and where \(P' = (x_0^\prime,\dotsc,x_N^\prime)\) is any other partition. We’ll show that \(U(f,P') \leq U(f,P)\text{;}\) the argument that \(L(f,P) \leq L(f,P')\) will be qualitatively identical.
    For each \(1 \leq k \leq N\text{,}\) let \(M_k^\prime \defn \sup\set{f(x)}{x_{k-1}^\prime \leq x \leq x_k^\prime}\text{,}\) so that \(U(f,P') = \sum_{k=1}^N M_k^\prime(x_k^\prime-x_{k-1}^\prime)\text{;}\) let \(M \defn \sup{f(x)}{a\leq x \leq b}\text{,}\) so that \(U(f,P) = M(b-a)\text{.}\) Given \(1 \leq k \leq \text{,}\) since \(f(x) \leq \sup{f(x)}{a\leq x \leq b} = M\) for all \(x \in [x_{k-1}^\prime,x_k^\prime]\text{,}\) it follows that
    \begin{align*} U(f,P') \amp \defn \sum_{k=1}^N M_k^\prime(x_k^\prime-x_k-1^\prime) \\ \amp \leq \sum_{k=1}^N M(x_k^\prime-x_{k-1}^\prime) \\ \amp = M\sum_{k=1}^N(x_k^\prime-x_{k-1}^\prime)\\ \amp = M(b-a) \\ \amp = U(f,P) \text{.} \end{align*}
    Finally, let us prove Item 3. Let \(P\) and \(P'\) be partitions of \([a,b]\text{.}\) Let \(P^{\prime\prime}\) be the partition of \([a,b]\) obtained by merging the lists \(P\) and \(P'\text{.}\) Then \(P^{\prime\prime}\) is a refinement of both \(P\) and \(P'\text{,}\) so that
    \begin{equation*} L(f,P) \leq L(f,P^{\prime\prime}) \leq U(f,P^{\prime\prime}) \leq U(f,P) \end{equation*}
    by Item 1 and Item 2.

    Definition 1.3.11.

    Let \(f\) be a bounded real-valued function on a closed interval \([a,b]\text{.}\)
    1. The lower integral of \(f\) on \([a,b]\) is
      \begin{equation*} \Lint{a}{b} f \defn \inf\set{L(f,P)}{P \text{ is a partition of }[a,b]}. \end{equation*}
    2. The upper integral of \(f\) on \([a,b]\) is
      \begin{equation*} \Uint{a}{b} f \defn \sup\set{L(f,P)}{P \text{ is a partition of }[a,b]}. \end{equation*}

    Theorem 1.3.12.

    Let \(f\) be a bounded real-valued function on a closed interval \([a,b]\text{.}\) Then its lower integral \(\Lint{a}{b}f\) and upper integral \(\Uint{a}{b}f\) both exist and satisfy
    \begin{equation*} \Lint{a}{b}f \leq \Uint{a}{b}f\text{.} \end{equation*}

    Proof.

    Let \(\cL(f)\) be the set of all lower sums of \(f\) with respect to partitions of \([a,b]\text{.}\) On the one hand, it is non-empty since it contains \(L(f,P)\) for \(P = (a,b)\text{.}\) On the other hand, if \(P\) is any partition of \([a,b]\text{,}\) then \(U(f,P)\) is an upper bound for \(\cL(f,P)\) by Item 3. Thus, \(\Lint{a}{b}f \defn \sup\cL(f)\) exists. A qualitatively identical argument shows that \(\Uint{a}{b}f = \sup\cU(f)\) exists, where \(\cU(f)\) is the set of all upper sums of \(f\) with respect to partitions of \([a,b]\text{.}\)
    Let us now show that \(\Lint{a}{b}f \leq \Uint{a}{b}f\text{.}\) Let \(P\) be any partition of \([a,b]\text{.}\) Recall that \(U(f,P)\) is an upper bound for \(\cL(f)\) by Item 3, so that
    \begin{equation*} U(f,P) \geq \sup\cL(f,P) = \Lint{a}{b}f\text{.} \end{equation*}
    However, this shows that \(\Lint{a}{b}f\) is a lower bound for \(\cU(f)\text{,}\) so that
    \begin{equation*} \Lint{a}{b}f \leq \inf\cU(f) = \Uint{a}{b}f\text{.} \end{equation*}

    Example 1.3.13.

    Let’s continue from Example 1.3.9. Recall that \(f\) is defined on \([0,1]\) by \(f(x) \defn x^2\text{.}\) On the one hand, for each \(N \in \N\text{,}\)
    \begin{equation*} \frac{1}{3} - \frac{1}{2N} + \frac{1}{6N^2} = L(f,P_N) \leq \Lint{0}{1} f\text{,} \end{equation*}
    so that
    \begin{equation*} \frac{1}{3} = \sup\set{\frac{1}{3} - \frac{1}{2N} + \frac{1}{6N^2}}{N \in \N} \leq \Lint{0}{1} f\text{.} \end{equation*}
    On the other hand, for each \(N \in \N\text{,}\)
    \begin{equation*} \frac{1}{3} + \frac{1}{2N} + \frac{1}{6N^2} = U(f,P_N) \geq \Uint{0}{1} f\text{,} \end{equation*}
    so that
    \begin{equation*} \frac{1}{3} = \inf\set{\frac{1}{3} + \frac{1}{2N} + \frac{1}{6N^2}}{N \in \N} \geq \Lint{0}{1} f\text{.} \end{equation*}
    Hence, by Theorem 1.3.12,
    \begin{equation*} \frac{1}{3} \leq \Lint{0}{1}f \leq \Uint{0}{1}f \leq \frac{1}{3}\text{,} \end{equation*}
    so that
    \begin{equation*} \Lint{0}{1}f = \frac{1}{3} = \Uint{0}{1}f\text{.} \end{equation*}
    This turns out to be a rigorous proof that the area of the planar region \(U\) of Theorem 1.1.1 is both well-defined and equal to \(\frac{1}{3}\text{.}\)
