    Section 1.5 A Non-Integrable Function

    A number \(x\) is rational if it can be written as \(x = \frac{p}{q}\) for \(p \in \Z\) and \(q \in \N\text{;}\) hence, let \(\Q\) denote the set of all rational numbers. Conversely, a number \(x\) is irrational if it isn’t rational; hence, let \(\Q^c\) denote the set of all irrational numbers, which reminds us that \(\Q^c\) is the complement of the subset \(\Q\) of \(\R\text{.}\)
    For example, we know that \(\sqrt{2} \in \Q^c\) (since the 5th century bce), that \(\eu \in \Q^c\) (since the 17th century), and \(\pi \in \Q^c\) (since the 18th century) However, somehow, we still don’t know whether \(\pi+\eu\text{,}\) \(\pi-\eu\text{,}\) and \(\pi\eu\) are irrational.
    The dichotomy between rational and irrational real numbers gives rise to the simplest example of a bounded but non-integrable function on a closed interval.

    Theorem 1.5.1.

    Let \(f\) be defined on \([0,1]\) by
    \begin{equation*} f(x) \defn \begin{cases}1 \amp \text{if }x\text{ is irrational,}\\ 0 \amp \text{else.}\end{cases} \end{equation*}
    Then \(\Lint{0}{1}f = 0\) and \(\Uint{0}{1}f = 1\text{,}\) so that \(f\) is not integrable on \([0,1]\text{.}\)
    How on earth do we prove a claim like this? We’ll use a fundamental property of the sets \(\Q\) and \(\Q^c\text{.}\)

    Definition 1.5.2.

    A set of real numbers \(A\) is dense if, for all \(a \lt b\text{,}\) there exists \(x \in A\) such that \(a \lt x \lt b\text{.}\)

    Remark 1.5.3.

    A set of numbers \(A\) is dense if and only if every number \(t\) can be approximated to any desired precision by some number in \(A\) (precisely, for every \(\epsilon \lt 0\text{,}\) no matter how small, there exists \(x \in A\) such that \(t-\epsilon \lt x \lt t+\epsilon\)).

    Proposition 1.5.4.

    The set \(\Q\) of all rational numbers is dense.
    The proof of this proposition requires two preparatory lemmas (or lemmata if you prefer the Greek plural).

    Lemma 1.5.5.

    Let \(x\) be a real number. There is a largest integer \(N\) such that \(N \leq x\text{.}\)

    Proof.

    Let \(A \defn \set{n \in \Z}{n \leq x}\text{.}\) On the one hand, \(A\) is non-empty:
    • if \(x \geq 0\text{,}\) then \(0 \in A\text{;}\)
    • if \(x \lt 0\text{,}\) then there exists \(k \in \N\) such that \(k \gt -x\) by Theorem 1.2.13, so that \(-k \lt x\text{,}\) and hence \(-k \in A\text{.}\)
    On the other hand, \(A\) is bounded from above by \(x\text{.}\) Thus, \(\alpha \defn \sup A\) is well-defined and \(\alpha \leq x\text{.}\)
    Now, since \(\alpha-1 \lt \alpha = \sup A\text{,}\) the number \(\alpha-1\) cannot be an upper bound for \(A\text{,}\) so there exists \(N \in A\) such that \(N \gt \alpha - 1\text{;}\) hence, let \(B \defn \set{n \in \Z}{n \leq N}\text{.}\) We’ll show that \(A = B\text{,}\) which will immediately imply that \(N = \alpha\) is the largest element of \(A\text{.}\) We’ll do so by showing that every element of \(A\) is an element of \(B\) and vice versa.
    Let us first show that every element of \(A\) is an element of \(B\text{;}\) we’ll do so using proof by contradiction. Let \(m \in A\) be given, and assume for the sake of argument that \(m \notin B\text{,}\) so that \(m \gt N\text{.}\) Then \(m \geq N+1 \gt (\alpha-1)+1 = \alpha\text{,}\) which is impossible since \(\alpha\) is an upper bound for \(A\text{.}\)
    Let us now show that every element of \(B\) is an element of \(A\text{.}\) Let \(m \in B\) be given. Then \(m \leq N\) by definition of \(B\) and \(N \leq x\) since \(N \in A\text{,}\) so that \(m \leq N \leq x\text{,}\) which shows that \(m \in A\text{.}\)

    Lemma 1.5.6.

    Let \(x\) and \(y\) be real numbers satisfying \(y-x \gt 1\text{.}\) There exists \(k \in \Z\) such that \(x \lt k \lt y\text{.}\)

    Proof.

    By Lemma 1.5.5, let \(N\) be the largest integer such that \(N \leq x\text{;}\) hence, set \(k \defn N+1\text{.}\) On the one hand, since \(N = \max\set{n \in \Z}{n \leq x}\) and \(k \gt N\text{,}\) it follows that \(k \gt x\text{.}\) On the other hand, since \(y-x \gt 1\) and \(N \leq x\text{,}\) it follow that \(y \gt 1+x \geq 1+N = k\text{.}\) Thus, we see that \(x \lt k \lt y\text{.}\)

    Proof of Proposition 1.5.4.

    Let \(x\) and \(y\) be real numbers satisfying \(x \lt y\text{.}\) By Corollary 1.2.14, there exists \(q \in \N\) such that \(\frac{1}{q} \lt y-x\text{.}\) Then \(qy-qx \gt 1\text{,}\) so, by Lemma 1.5.6, there exists \(p \in \Z\) such that \(qx \lt p \leq qy\text{.}\) Then \(r \defn \frac{p}{q} \in \Q\) satisfies \(x \lt r \lt y\text{.}\)

    Proposition 1.5.7.

    The set \(\Q^c\) of all irrational numbers is dense.
    Again, we’ll two preparatory lemmas (or lemmata if you prefer).

    Lemma 1.5.8.

    Let \(r\) and \(s\) be rational numbers such that \(r \lt s\text{.}\) There exists irrational \(w\) such that \(r \lt w \lt s\text{.}\)

    Proof.

    By bringing \(r\) and \(s\) over a common denominator, we can write \(r = \frac{p_1}{q}\) and \(s = \frac{p_2}{q}\) for integers \(p_1 \lt p_2\) and \(q \in \N\text{.}\) Hence, let \(w \defn r+\frac{\sqrt{2}}{2q}\text{.}\) We’ll show that \(w\) is irrational and that \(r \lt w \lt s\text{.}\)
    Let’s first show that \(w\) is irrational; we’ll prove this by contradiction. So, assume for the sake of argument that \(w\) is rational, so \(r = \frac{p'}{q'}\) for some \(p' \in \Z\) and \(q \in \N\text{.}\) Then \(\frac{2p_1 + \sqrt{2}}{2q} = r = \frac{p'}{q'}\text{,}\) so that \(\sqrt{2} = \frac{2qp'}{q'} - 2p_1\) is rational, which contradicts the fact (known to the ancient Greeks!) that \(\sqrt{2}\) is irrational.
    Let’s now show that \(r \lt w \lt s\text{.}\) On the one hand, sicne \(\sqrt{2} \lt 0\text{,}\) it follows that \(w = r + \frac{\sqrt{2}}{2q} \gt r\) On the other hand, since the square root function is strictly increasing on \((0,\infty)\text{,}\) it follows that \(\sqrt{2} \lt \sqrt{4} = 2\text{,}\) so that \(\frac{\sqrt{2}}{2q} \lt \frac{1}{q}\text{,}\) and hence
    \begin{equation*} w = r + \frac{\sqrt{2}}{2q} \lt r + \frac{1}{q} = \frac{p_1 + 1}{q} \leq \frac{p_2}{q} = s. \end{equation*}
    Thus, we see that \(r \lt w \lt s\text{.}\)

    Proof of Proposition 1.5.7.

    Let \(x\) and \(y\) be real numbers such that \(x \lt y\text{.}\) First, by Proposition 1.5.4, there exists rational \(r\) such that \(x \lt r \lt y\text{.}\) Next, by Proposition 1.5.4, there exists rational \(s\) such that \(r \lt s \lt y\text{.}\) Finally, by Lemma 1.5.8, there exists irrational \(w\) such that \(r \lt w \lt s\text{.}\) Thus, at last,
    \begin{equation*} x \lt r \lt w \lt s \lt y\text{,} \end{equation*}
    so that \(x \lt w \lt y\text{.}\)
    We now have everything we need to prove non-integrability of the function \(f\text{.}\)

    Proof of Theorem 1.5.1.

    Let \(P = (x_0,x_1,\dotsc,x_N)\) be any partition of \([0,1]\text{.}\) Let \(1 \leq k \leq N\) be given. On the one hand, by Proposition 1.5.7, there exists irrational \(w_k\) such that \(x_{k-1} \lt w_k \lt x_k\text{,}\) so that \(f(w_k) = 0\text{,}\) and hence
    \begin{equation*} m_k = \inf\set{f(x)}{x_{k-1}\leq x \leq x} = \min\set{f(x)}{x_{k-1}\leq x \leq x} = 0\text{.} \end{equation*}
    On the other hand, by Proposition 1.5.7, there exists rational \(r_k\) such that \(x_{k-1} \lt r_k \lt x_k\text{,}\) so that \(f(r_k) = 1\text{,}\) and hence
    \begin{equation*} M_k = \sup\set{f(x)}{x_{k-1}\leq x \leq x} = \max\set{f(x)}{x_{k-1}\leq x \leq x} = 1\text{.} \end{equation*}
    Thus
    \begin{equation*} L(f,P) = \sum_{k=1}^N m_k(x_k-x_{k-1})) = \sum_{k=1}^N 0(x_k-x_{k-1}) = 0 \end{equation*}
    and
    \begin{equation*} U(f,P) = \sum_{k=1}^N M_k(x_k-x_{k-1}) = \sum_{k=1}^N (x_k-x_{k-1}) = 1-0 = 1\text{.} \end{equation*}
    Since the partition \(P\) was arbitrary, it follows that
    \begin{equation*} \Lint{0}{1} f = \sup\cL(f) = \sup\{0\} = 0 \end{equation*}
    and
    \begin{equation*} \Uint{0}{1} f = \inf\cU(f) = \inf\{1\} = 1\text{.} \end{equation*}
    In particular, since \(\Lint{0}{1}f \neq \Uint{0}{1}f\text{,}\) we see that \(f\) is not integrable.
