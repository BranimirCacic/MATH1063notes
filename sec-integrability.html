    Section 1.4 Integrability and integration

    In this section, we give a more-or-less rigorous alternative to Section 5.4
     1 
    opentext.uleth.ca/apex-calculus/sec_riemann.html
    of APEX Calculus and Section 5.5 of Strang’s Calculus. Thus, instead of hand-waving around Riemann’s original formulation of a rigorous theory of integration, we give a fairly precise account using technical simplifications due to Darboux. Thus, in this course, we’re technically considering Darboux integration instead of Riemann integration sensu stricto, but the two theories turn out to be equivalent
     2 
    proofwiki.org/wiki/Equivalence_of_Definitions_of_Riemann_and_Darboux_Integrals
    .

    Definition 1.4.1.

    Let \(f\) be a bounded function on a closed interval \([a,b]\text{.}\) We say that \(f\) is integrable on \([a,b]\) whenever \(\Lint{a}{b}f = \Uint{a}{b}f\text{,}\) in which case the integral of \(f\) on \([a,b]\) is
    \begin{equation*} \int_a^b f \defn \Lint{a}{b}f = \Uint{a}{b}f\text{.} \end{equation*}

    Note 1.4.2.

    Leibniz’s notation for \(\int_a^b f\) is \(\int_a^b f(x) \du{x}\text{.}\) This notation conveys the intuition that the integral \(\int_a^b f\) is a “continuous sum” over \(a \leq x \leq b\) of terms \(f(x)\du{x}\text{,}\) where \(\du{x}\) is the “width” of an “infinitesimal subinterval” of \([a,b]\) (note the scare quotes).

    Note 1.4.3.

    Suppose that \(f\) is a continuous function on \([a,b]\text{.}\) Then \(f\) is bounded by the extreme value theorem
     3 
    opentext.uleth.ca/apex-calculus/sec_extreme_values.html#thm_extreme_val
    . In particular, since \(f\) attains both a maximum and minimum on \([a,b]\text{,}\) it follows that
    \begin{align*} \inf\set{f(x)}{a \leq x \leq b} \amp = \min\set{f(x)}{a \leq x \leq b} \\ \sup\set{f(x)}{a \leq x \leq b} \amp = \max\set{f(x)}{a \leq x \leq b} \text{.} \end{align*}

    Example 1.4.4.

    Define \(f\) on \([0,1]\) by \(f(x) := x^2\text{.}\) Since \(\Lint{0}{1}f = \frac{1}{3} = \Uint{0}{1}f\) by Example 1.3.13, it follows that \(f\) is integrable with integral \(\int_0^1 f = \frac{1}{3}\) (i.e., \(\int_0^1 x^2 \du{x} = \frac{1}{3}\)). This turns out to yield a rigorous proof that the region \(U\) of Theorem 1.1.1 has a well-defined area equal to \(\frac{1}{3}\text{.}\)

    Proposition 1.4.5.

    Let \(f\) be a constant function on \([a,b]\text{,}\) and let \(c\) denote the constant value of \(f\text{,}\) i.e., \(f(x) = c\) for all \(a \leq x \leq b\text{.}\) Then \(\int_a^b f = c(b-a)\) (i.e., \(\int_a^b c \du{x} = c(b-a)\)).

    Proof.

    Let \(P = (x_0,x_1,\dotsc,x_N)\) be any partition of \([a,b]\text{.}\) Since \(f\) has the constant value \(c\text{,}\) it follows that
    \begin{align*} m_k \amp \defn \inf\set{f(x)}{x_{k-1} \leq x \leq x_k} = \inf\{c\} = c\\ M_k \amp \defn \sup\set{f(x)}{x_{k-1} \leq x \leq x_k} = \inf\{c\} = c \end{align*}
    for all \(1 \leq k \leq N\text{.}\) Hence, by Remark 1.3.4, it follows that
    \begin{align*} L(f,P) \amp \defn \sum_{k=1}^N m_k(x_k-x_{k-1}) = \sum_{k=1}^N c(x_k-x_{k-1}) = c(b-a)\\ U(f,P) \amp \defn \sum_{k=1}^N M_k(x_k-x_{k-1}) = \sum_{k=1}^N c(x_k-x_{k-1}) = c(b-a)\text{.} \end{align*}
    Thus, at last,
    \begin{align*} \Lint{a}{b}f \amp =\sup\cL(f) = \sup\{c(b-a)\} = c(b-a)\\ \Uint{a}{b}f \amp =\inf\cU(f) = \inf\{c(b-a)\} = c(b-a) \end{align*}
    so that \(f\) is integrable and \(\int_a^b f = c(b-a)\text{.}\)
    While the above definition of integrability is most closely linked to the geometric intuition of approximating the area under the graph of a function from above and below, it generally quite difficult to prove. Instead, one uses the following necessary and sufficient condition.

    Theorem 1.4.6.

    Let \(f\) be a bounded function on \([a,b]\text{.}\) The following are equivalent:
    1. the function \(f\) is integrable on \([a,b]\text{;}\)
    2. for every \(\epsilon \gt 0\text{,}\) there exists a partition \(P\) of \([a,b]\) such that \(U(f,P) - L(f,P) \lt \epsilon\text{.}\)

    Proof.

    Remark 1.4.7.

    A detailed proof of Theorem 1.4.6 can be found here
     4 
    proofwiki.org/wiki/Condition_for_Darboux_Integrability
    . Be aware of the following quirks of notation and terminology:
    • a closed interval \([a,b]\) is denoted by \([a \,.\,.\, b]\text{;}\)
    • a partition is called a finite subdivision;
    • the lower sum \(L(f,P)\)of a function \(f\) with respect to a partition \(P\) of an interval \([a,b]\) is denoted by \(L(P)\text{,}\) and \(\sup\cL(f)\) is denoted by \(\sup_P L(P)\text{;}\)
    • the upper sum \(U(f,P)\)of a function \(f\) with respect to a partition \(P\) of an interval \([a,b]\) is denoted by \(U(P)\text{,}\) and \(\inf\cU(f)\) is denoted by \(\inf_P U(P)\text{.}\)

    Example 1.4.8.

    Define a function \(f\) on \([0,1]\) by
    \begin{equation*} f(x) \defn \begin{cases}1 \amp \text{if } x = 1\\ 0 \amp \text{else}\end{cases}\text{.} \end{equation*}
    Then \(f\) is integrable on \([0,2]\) and \(\int_0^2 f = 0\text{.}\)
    Let’s first show that \(f\) is integrable using Theorem 1.4.6. Let \(0 \lt \epsilon \lt 4\) be given. Define a partition \(P_\epsilon = (x_0,x_1,x_2,x_3)\) of \([0,2]\) by
    \begin{align*} x_0 \amp \defn 0 \amp x_1 \amp \defn 1-\frac{\epsilon}{4} \amp x_2 \amp \defn 1+\frac{\epsilon}{4} \amp x_3 \amp \defn 2 \text{.} \end{align*}
    On the one hand, on both \([x_0,x_1]\) and \([x_2,x_3]\text{,}\) the function \(f\) has the constant value \(0\text{,}\) so that \(m_1 = 0 = M_1\) and \(m_3 = 0 = M_3\text{.}\) On the other hand, on \([x_1,x_2]\text{,}\) the function \(f\) attains a minimum of \(0\) and a maximum of \(1\text{,}\) so that \(m_2 = 0\) and \(M_2 = 1\text{.}\) Hence,
    \begin{equation*} L(f,P_\epsilon) = \sum_{k=1}^3 m_k (x_k-x_{k-1}) = \sum_{k=1}^3 0(x_k-x_{k-1}) = 0 \end{equation*}
    and
    \begin{align*} U(f,P_\epsilon) \amp = \sum_{k=1}^3 M_k (x_k-x_{k-1})\\ \amp = 0 \cdot ((1-\tfrac{\epsilon}{4})-1) + 1 \cdot ((1+\tfrac{\epsilon}{4})-(1-\tfrac{\epsilon}{4})) + 0 \cdot (2-(1+\tfrac{\epsilon}{4})) \\ \amp = \frac{\epsilon}{2} \text{,} \end{align*}
    so that
    \begin{equation*} U(f,P_\epsilon) - L(f,P_\epsilon) = \frac{\epsilon}{2}-0 = \frac{\epsilon}{2} \lt \epsilon. \end{equation*}
    Since we can do this for all sufficienly small \(\epsilon \gt 0\text{,}\) it follows that \(f\) is integrable on \([0,2]\text{.}\)
    Let us now compute \(\int_0^2 f\text{.}\) By our computations above, for every \(0 \lt \epsilon \lt 4\text{,}\)
    \begin{equation*} 0 = L(f,P_\epsilon) \leq \Lint{0}{2}f = \int_0^2 f = \Uint{0}{2} f \leq U(f,P_\epsilon) = \frac{\epsilon}{2} \lt \epsilon\text{,} \end{equation*}
    so that \(0 \leq \int_0^2 f \lt \epsilon\) for all sufficiently small \(\epsilon \gt 0\text{.}\) Thus, \(\int_0^2 f = 0\text{.}\)

    Remark 1.4.9.

    In Example 1.4.8, we concluded that two numbers \(a\) and \(b\) were equal because \(\abs{a-b} \leq \epsilon\) for all \(\epsilon \gt 0\text{.}\) This is a typical way to prove an equality in analysis. In fact, given real numbers \(a\) and \(b\text{,}\) the following statements are equivalent:
    1. \(a \leq b\text{;}\)
    2. \(a \leq b + \epsilon\) for every \(\epsilon \gt 0\text{;}\)
    3. \(a-\epsilon \leq b\) for every \(\epsilon \gt 0\text{.}\)
    We now need to know that the kinds of functions we typically work with in calculus are actually integrable on closed intervals. Using the theory of uniform continuity, one can actually show that a continuous function \(f\) on a closed interval \([a,b]\) is necessarily integrable. However, the following special case will suffice for our purposes.

    Theorem 1.4.10.

    Let \(f\) be continuously differentiable on \([a,b]\text{.}\) Then \(f\) is integrable on \([a,b]\text{.}\)

    Proof.

    As a preliminary step, we show that there exists a constant \(C \gt 0\) such that \(\abs{f(t)-f(s)} \leq C\abs{s-t}\) for all \(s,t \in [a,b]\text{.}\) Since the derivative \(f'\) is continuous on \([a,b]\text{,}\) it is bounded on \([a,b]\) by the extreme value theorem
     5 
    opentext.uleth.ca/apex-calculus/sec_extreme_values.html#thm_extreme_val
    . Hence, let \(C \defn \sup\set{\abs{f'(x)}}{a \leq x \leq b} + 1\) to ensure that \(C \gt 0\text{.}\) Now, let \(s,t \in [a,b]\) be given; without loss of generality, we may assume that \(a \leq s \lt t \leq b\text{.}\) Then, by the mean value theorem
     6 
    opentext.uleth.ca/apex-calculus/sec_mvt.html#thm_mvt
    , there exists some \(s \lt c \lt t\) such that \(f(t)-f(s) = f'(c)(t-s)\text{.}\) Thus,
    \begin{equation*} \abs{f(t)-f(s)} = \abs{f'(c)(t-s)} = \abs{f'(c)}\abs{s-t} \leq C\abs{s-t}\text{.} \end{equation*}
    We now apply the integrability criterion of Theorem 1.4.6. Let \(\epsilon \gt 0\) be given. By Corollary 1.2.14, find \(N \in \N\) such that \(\frac{1}{N} \lt \frac{\epsilon}{2C(b-a)^2}\text{.}\) Hence, let \(P_N = (x_0,\dotsc,x_N)\) be the partition of \([a,b]\) constructed in Example 1.3.2, so that
    \begin{equation*} \abs{x_k-x_{k-1}} = x_k-x_{k-1} = \frac{b-a}{N} \end{equation*}
    for all \(1 \leq k \leq N\text{.}\) Thus, by our estimate above, given \(1 \leq k \leq N\text{,}\) we find that
    \begin{align*} f(s) - f(t) \leq \amp \abs{f(s)-f(t)}\\ \amp\leq C\abs{s-t} \leq C \cdot \frac{b-a}{N} \\ \amp \lt C(b-a) \cdot \frac{\epsilon}{2C(b-a)^2}\\ \lt \frac{\epsilon}{2(b-a)} \end{align*}
    for all \(s,t \in [x_{k-1},x_k]\text{.}\)
    Now, let \(1 \leq k \leq N\) be given. Given \(x_{k_1} \leq t \leq x_k\text{,}\) it follows that \(f(s) \leq f(t) + \frac{\epsilon}{2(b-a)}\) for all \(x_{k-1} \leq s \leq x_k\text{,}\) so that \(M_k \leq f(t) + \frac{\epsilon}{2(b-a)}\text{.}\) But now, we see that \(M_k - \frac{\epsilon}{2(b-a)} \leq f(t)\) for all \(x_{k-1} \leq t \leq x\text{,}\) so that \(M_k - \frac{\epsilon}{2(b-a)} \leq m_k\text{,}\) i.e.,
    \begin{equation*} M_k - m_k \leq \frac{\epsilon}{2(b-a)}\text{.} \end{equation*}
    Thus, at last, we find that
    \begin{align*} U(f,P) - L(f,P)\amp \sum_{k=1}^N M_k(x_k-x_{k-1}) - \sum_{k=1}^N m_k(x_k-x_{k-1}) \\ \amp = \sum_{k=1}^N (M_k-m_k)(x_k-x_{k-1}) \\ \amp \leq \sum_{k=1}^N \frac{\epsilon}{2(b-a)}(x_k-x_{k-1})\\ \amp = \frac{\epsilon}{2(b-a)}\sum_{k=1}^N (x_k-x_{k-1})\\ \amp = \frac{\epsilon}{2(b-a)}(b-a)\\ \amp = \frac{\epsilon}{2} \\ \amp \lt \epsilon \text{.} \end{align*}
    Finally, we record some important basic properties of the integral.

    Proposition 1.4.11. Basic properties of integration.

    1. Suppose that \(f\) is integrable on \([a,b]\) and that \(a \lt c \lt b\text{.}\) Then \(f\) is integrable on both \([a,c]\) and \([c,b]\) and
      \begin{equation*} \int_a^b f = \int_a^c f + \int_c^b f\text{,} \end{equation*}
      i.e., \(\int_a^b f(x)\du{x} = \int_a^c f(x)\du{x} + \int_c^b f(x)\du{x}\text{.}\)
    2. Suppose that \(f\) and \(g\) are both integrable on \([a,b]\text{.}\) Define \(f+g\) on \([a,b]\) by \((f+g)(x) \defn f(x) + g(x)\text{.}\) Then \(f+g\) is integrable on \([a,b]\) and
      \begin{equation*} \int_a^b (f+g) = \int_a^b f + \int_a^b g \end{equation*}
      i.e., \(\int_a^b \left(f(x)+g(x)\right)\du{x} = \int_a^b f(x)\du{x} + \int_a^b g(x)\du{x}\text{.}\)
    3. Suppose that \(f\) is integrable on \([a,b]\text{;}\) let \(c\) be a real number. Define \(cf\) on \([a,b]\) by \((cf)(x) \defn cf(x)\text{.}\) Then \(cf\) is integrable on \([a,b]\) and
      \begin{equation*} \int_a^b cf = c \int_a^b f \end{equation*}
      i.e., \(\int_a^b c f(x) \du{x} = c \int_a^b f(x)\du{x}\text{.}\)

    Proof.

    I’ll give the proof of Item 2 as a typical example of how these kinds of facts are proved (and an illustration of how abjectly tedious the necessary bookkeeping is). In what follows, if \(h\) is an integrable function on \([a,b]\) and \(P = (x_0,x_1,\dotsc,x_N)\) is a partition of \([a,b]\text{,}\) let
    \begin{align*} m_k[h,P] \amp \defn \inf\set{h(x)}{x_{k_1}\leq x \leq x_k}\\ M_k \amp \defn \sup\set{h(x)}{x_{k_1}\leq x \leq x_k} \end{align*}
    for each \(1 \leq k \leq N\text{,}\) so that
    \begin{align*} L(h,P) \amp = \sum_{k=1}^N m_k[h,P](x_k-x_{k-1}) \\ U(h,P) \amp = \sum_{k=1}^N M_k[h,P] (x_k-x_{k-1})\text{.} \end{align*}
    Let’s first prove that \(f+g\) is integrable. Let \(\epsilon \gt 0\) be given. On the one hand, by Theorem 1.4.6 applied to \(f\text{,}\) there exists a partition \(P_1\) of \([a,b]\) such that \(U(f,P_1) - L(f,P_1) \lt \frac{\epsilon}{2}\text{.}\) On the other hand, by Theorem 1.4.6 applied to \(g\text{,}\) there exists a partition \(P_2\) of \([a,b]\) such that \(U(g,P_2) - L(f,P_2) \lt \frac{\epsilon}{2}\text{.}\) Hence, let \(P=(x_0,x_1,\dotsc,x_N)\) be the partition of \([a,b]\) obtained by merging the lists \(P_1\) and \(P_2\text{,}\) so that, by Item 2,
    \begin{align*} L(f,P) \amp \geq L(f,P_1) \amp U(f,P) \amp \leq U(f,P_1)\\ L(g,P) \amp \geq L(g,P_2) \amp U(g,P) \amp \leq U(g,P_2)\text{.} \end{align*}
    Now, for each \(1 \leq k \leq N\text{,}\) observe that
    \begin{equation*} m_k[f,P]+m_k[g,P] \leq f(x) + g(x) \leq M_k[f,P]+M_k[g,P] \end{equation*}
    for all \(x_{k-1} \leq x \leq x_k\text{,}\) so that
    \begin{align*} m_k[f,P] + m_k[g,P] \amp \leq m_k[f+g,P]\\ M_k[f,P] + M_k[g,P] \amp \geq M_k[f+g,P]\text{.} \end{align*}
    Hence,
    \begin{align*} L(f+g,P) \amp = \sum_{k=1}^N m_k[f+g,P](x_k-x_{k-1}) \\ \amp \geq \sum_{k=1}^N \left(m_k[f,P]+m_k[g,P]\right)(x_k-x_{k-1}) \\ \amp = \sum_{k=1}^N m_k[f,P](x_k-x_{k-1}) + \sum_{k=1}^N m_k[g,P](x_k-x_{k-1})\\ \amp = L(f,P) + L(g,P) \\ \amp \geq L(f,P_1) + L(g,P_2) \text{.} \end{align*}
    Similarly, one can show that \(U(f+g,P) \leq U(f,P_1) + U(g,P_2)\text{.}\) Thus, at last,
    \begin{align*} U(f+g,P) - L(f+g,P) \amp \leq \left(U(f,P_1) + U(g,P_2)\right) - \left(L(f,P_1) + L(g,P_2)\right) \\ \amp = \left(U(f,P_1)-L(f,P_1)\right) + \left(U(g,P_2) - L(g,P_2)\right) \\ \amp \lt \frac{\epsilon}{2} + \frac{\epsilon}{2}\\ \amp = \epsilon \text{.} \end{align*}
    Let’s now show that \(\int_a^b(f+g) = \int_a^b f + \int_a^b g\text{.}\) Let \(\epsilon \gt 0\) be given. By our constructions above, there exists a partition \(P\) of \([a,b]\) such that
    \begin{align*} U(f,P) - L(f,P) \amp \lt \frac{\epsilon}{4} \amp U(g,P) - L(g,P) \amp \lt \frac{\epsilon}{4} \text{,} \end{align*}
    and moreover,
    \begin{align*} L(f,P) + L(g,P) \amp \leq L(f+g,P) \amp U(f+g,P) \amp \leq U(f,P) + U(g,P)\text{.} \end{align*}
    On the one hand,
    \begin{equation*} L(f,P) + L(g,P) \leq \int_a^b (f+g) \leq U(f,P) + U(g,P) \end{equation*}
    since
    \begin{equation*} L(f,P) + L(g,P) \leq L(f+g,P) \leq \Lint{a}{b}(f+g) = \int_a^b (f+g) \end{equation*}
    and
    \begin{equation*} U(f,P) + U(g,P) \geq U(f+g,P) \geq \Uint{a}{b}(f+g) = \int_a^b(f+g)\text{.} \end{equation*}
    On the other hand,
    \begin{equation*} L(f,P) + L(g,P) \leq \int_a^b f + \int_a^b g \leq U(f,P) + U(g,P) \end{equation*}
    since
    \begin{equation*} L(f,P) + L(g,P) \leq \Lint{a}{b}f + \Lint{a}{b} g = \int_a^b f + \int_a^b g \end{equation*}
    and
    \begin{equation*} U(f,P) + U(g,P) \geq \Uint{a}{b}f + \Uint{a}{b}g = \int_a^b f = \int_a^b g. \end{equation*}
    Thus, we find that
    \begin{align*} 0 \amp \leq \left(\int_a^b (f+g)) \right) - \left(L(f,P)+L(g,P)\right) \\ \amp \leq \left(U(f,P)+U(g,P)\right) - \left(L(f,P)+L(g,P)\right) \\ \amp = \left(U(f,P)-L(f,P)\right) + \left(U(g,P) - L(g,P)\right) \\ \amp = \frac{\epsilon}{4} + \frac{\epsilon}{4} \\ \amp = \frac{\epsilon}{2}\text{,} \end{align*}
    and similarly
    \begin{equation*} 0 \leq \left(\int_a^b f + \int_a^b g \right) - \left(L(f,P)+L(g,P)\right) \lt \frac{\epsilon}{2}\text{.} \end{equation*}
    Hence, at last, by the triangle inequality,
    \begin{align*} \abs{\left(\int_a^b f + \int_a^b g \right) - \int_a^b(f+g)} \amp \leq \abs{\left(\int_a^b f + \int_a^b g \right) - \left(L(f,P)+L(g,P)\right)}\\ \amp \quad \abs{\int_a^b (f+g) - \left(L(f,P)+L(g,P)\right)}\\ \amp \leq \frac{\epsilon}{2} + \frac{\epsilon}{2} \\ \amp =\epsilon\text{.} \end{align*}
    Since \(\epsilon \gt 0\) was arbitrary, it follows that \(\int_a^b f + \int_a^b g = \int_a^b(f+g)\text{.}\)

    Example 1.4.12.

    Let’s find \(\int_0^1(4+3x^2)\du{x}\text{,}\) i.e., \(\int_0^1 f\) for \(f(x) \defn 4+3x^2\text{.}\)
    If we’re being hyper-pedantic, we’ll proceed as follows. Note that \(f = g+3h\) for \(g(x) \defn 4\) and \(h(x) = x^2\text{.}\) On the one hand, by Proposition 1.4.5, the constant function \(g\) is integrable on \([0,1]\) and
    \begin{equation*} \int_0^1 g = 4(1-0) = 4\text{.} \end{equation*}
    On the other hand, by Example 1.4.4, the function \(h\) is integrable on \([0,1]\) and \(\int_0^1 h = \frac{1}{3}\text{,}\) so that, by Proposition 1.4.11, the function \(3h\) is integrable on \([0,1]\) and
    \begin{equation*} \int_0^1 3h = 3\int_0^1 h = 3 \cdot \frac{1}{3} = 1\text{.} \end{equation*}
    Thus, by Proposition 1.4.11, the function \(f = g+3h\) is also integrable on \([0,1]\) and
    \begin{equation*} \int_0^1 f = \int_0^1 (g+3h) = \int_0^1 g + \int_0^1 3h = 4 + 1 = 5\text{.} \end{equation*}
    In practice, one proceeds as follows. Since
    \begin{equation*} \int_0^1 4 \du{x} = 4(1-0) = 4 \end{equation*}
    and \(\int_0^1 x^2\du{x} = \frac{1}{3}\text{,}\) it follows (i.e. by repeated applications of Proposition 1.4.11) that
    \begin{equation*} \int_0^1 (4+3x^2)\du{x} = \int_0^1 4\du{x} = 3\int_0^1x^2\du{x} = 4 + 3 \cdot \frac{1}{3} = 5\text{.} \end{equation*}
